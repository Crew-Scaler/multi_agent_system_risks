# 10.6.22 Model Registry Version Selection Manipulation for Cross-Agent Compromise [NoFW]

MLflow model registry enables agents to select model versions by semantic constraints. Attackers compromising the registry inject malicious versions matching version constraints, causing all agents querying the registry to simultaneously load compromised models. Agent A loading the compromised version passes results to Agent B which loads the same registry entry, creating coordinated multi-agent compromise.
