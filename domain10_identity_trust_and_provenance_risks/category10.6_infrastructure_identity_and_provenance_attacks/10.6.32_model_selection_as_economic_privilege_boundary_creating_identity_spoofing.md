# 10.6.32 Model Selection as Economic Privilege Boundary Creating Identity Spoofing [NoFW]

Model selection creates implicit capability tiers where different models represent different privilege levels. In multi-agent systems, attackers spoof agent identity by compromising agent API calls to invoke higher-tier models while appearing to use lower-capability ones, or vice versa, misrepresenting capability boundaries to deceive trust evaluation.
