# 10.6.8 Engine Binary Non-Fungibility Breaking Agent Identity [NoFW]

TensorRT engines compiled for specific GPUs are non-fungible—two engines for the same model but different GPU architectures produce different results. An attacker migrating an agent to different hardware changes its engine, effectively changing its identity without changing its agent ID. Multi-agent systems where agent identity is abstracted from hardware variants create identity spoofing through hardware migration.
