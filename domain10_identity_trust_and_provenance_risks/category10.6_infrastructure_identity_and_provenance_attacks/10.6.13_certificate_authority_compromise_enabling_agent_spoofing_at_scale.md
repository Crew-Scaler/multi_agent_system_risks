# 10.6.13 Certificate Authority Compromise Enabling Agent Spoofing at Scale [NoFW]

Fleet Command uses a centralized certificate authority to issue certificates to edge agents. An attacker compromising the certificate authority can issue certificates for arbitrary agent identities, creating fake agents joining the fleet that impersonate real agents. Multi-agent Fleet Command systems where agents authenticate to each other create systematic agent spoofing risk when the shared CA is compromised.
