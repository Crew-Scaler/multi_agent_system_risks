# 10.6.14 MLflow Model Registry Identity Spoofing Through Version Tagging and Range Ambiguity [NoFW]

Model registries using semantic versioning create identity spoofing through version tags and range matching where attackers register malicious versions appearing as legitimate updates. Shared registries create spoofing impacts across all agents. Multi-agent systems using shared registry tags or version range queries (e.g., "^1.2.0") enable attacker versions to pass validation and reach all dependent agents simultaneously.
