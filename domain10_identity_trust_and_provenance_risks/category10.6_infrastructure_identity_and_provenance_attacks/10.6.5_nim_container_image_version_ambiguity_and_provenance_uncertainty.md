# 10.6.5 NIM Container Image Version Ambiguity and Provenance Uncertainty [NoFW]

NIM image references relying on floating tags (e.g., "latest") become non-deterministic when attackers compromise image registries and replace the tag with backdoored versions. All agents auto-pulling the floating tag transparently deploy compromised versions. Unlike singular deployments with pinned versions, multi-agent deployments using floating tags create provenance ambiguity enabling scale compromise through registry manipulation.
