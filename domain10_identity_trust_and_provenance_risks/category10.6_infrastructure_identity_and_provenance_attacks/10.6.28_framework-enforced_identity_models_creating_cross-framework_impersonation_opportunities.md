# 10.6.28 Framework-Enforced Identity Models Creating Cross-Framework Impersonation Opportunities [NoFW]

Different frameworks implement agent identity differently (LangChain's implicit identity, LangGraph's node identity, AutoGen's instance names, CrewAI's role-based identity, Semantic Kernel's plugin identity). Multi-agent systems integrating across frameworks create identity translation gaps where N frameworks create N(N-1) translation semantics likely containing impersonation opportunities.
