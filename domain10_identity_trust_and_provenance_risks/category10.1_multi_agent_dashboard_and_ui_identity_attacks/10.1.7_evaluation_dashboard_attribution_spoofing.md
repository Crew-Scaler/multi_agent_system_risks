# 10.1.7 Evaluation Dashboard Attribution Spoofing [NoFW]

Evaluation dashboards displaying results from multiple agents could attribute malicious outputs to trusted agents when attribution relies on self-reported metadata. Attackers have compromised agents emit results alongside forged metadata fields (agent_id, agent_role) that the dashboard renders as the identity of a trusted peer, causing downstream systems to act on falsely attributed evaluation data.
