# 10.1.5 Multi-Agent Dashboard Attribution Spoofing Through Visual Similarity [NoFW]

Dashboards use visual indicators (agent names, avatars, colors) to distinguish agent identities but rely on weak cryptographic binding. Compromised agents style malicious content to appear from trusted agents when identity derives from manipulable metadata fields (agent_name, agent_type), enabling users to execute attacker instructions without scrutiny.
