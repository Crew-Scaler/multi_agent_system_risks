# 10.1.6 Agent Impersonation Through Message Source Field Manipulation [NoFW]

Chat interfaces distinguish messages by source identifiers derived from metadata fields. When these fields are manipulable through prompt injection rather than cryptographically bound, attackers perform impersonation where malicious content appears from trusted agents by injecting instructions that cause compromised agents to emit messages with spoofed metadata. The UI renders malicious responses with trusted agent visual styling and authority badges.
