# 10.2.9 Checkpoint Attribution Ambiguity in Multi-Agent Workflows [NoFW]

LangGraph checkpoints maintain execution state but may lack clear attribution about which agent made specific state changes. In multi-agent workflows, determining which agent was responsible for corrupted state is difficult when checkpoints aggregate multiple agents' mutations without per-agent field attribution, creating forensic blind spots exploitable by attackers.
