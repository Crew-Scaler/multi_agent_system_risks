# 10.2.5 Approval Workflow Multi-Stage Provenance Gaps in Sequential Agent Reviews [NoFW]

Approval workflow UIs displaying final recommendations from multi-stage agent pipelines without cryptographic provenance for intermediate stages create accountability gaps. A recommendation resulting from Agent A passing to Agent B passing to Agent C may only show Agent C's output, allowing early-stage compromise to poison analysis while remaining forensically invisible.
