# 10.2.8 Knowledge Base Source Attribution Loss Through Multi-Agent Processing [NoFW]

Knowledge base documents track source but multi-agent processing loses provenance. Multi-hop agent chains progressively lose source attribution enabling instruction laundering where origin becomes untraceable. A retrieval agent fetches a document with source metadata, a downstream summarizer rewrites and strips citations, and a decision agent uses the summary with no ability to reconstruct the original source.
