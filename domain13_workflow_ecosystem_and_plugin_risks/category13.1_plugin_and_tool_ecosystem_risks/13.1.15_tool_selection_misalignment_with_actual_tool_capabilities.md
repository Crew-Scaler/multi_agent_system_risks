# 13.1.15 Tool Selection Misalignment With Actual Tool Capabilities [NoFW]

Agent tool registries contain declared capability descriptions that diverge from actual tool behavior over time as tools evolve while registry entries remain static. Attackers exploit this divergence by triggering agents to select tools based on outdated capability declarations, causing agent workflows to invoke tools that produce unexpected outputs mismatched with downstream expectations. In multi-agent systems sharing central tool registries, one stale registry entry propagates capability misalignment to all agents selecting tools from that registry.
