# 13.1.20 Profiling Hook Installation as Tool Behavior Modification [NoFW]

Performance profiling infrastructure (Python profilers, tracing hooks, instrumentation frameworks) installed on agent runtime environments can be abused as tool behavior modification vectors. Attackers with access to profiling configuration inject hooks that intercept tool function calls, modifying input parameters or output values before they reach the agent. Unlike prompt injection targeting model inputs, profiling hook attacks operate at the runtime layer, affecting tool behavior transparently regardless of model awareness. In multi-agent systems sharing instrumentation infrastructure, one poisoned profiling configuration modifies tool behavior across all agents in the shared environment.
