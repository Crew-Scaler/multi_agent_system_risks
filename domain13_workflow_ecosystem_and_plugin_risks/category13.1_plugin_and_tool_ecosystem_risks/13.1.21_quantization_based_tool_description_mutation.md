# 13.1.21 Quantization-Based Tool Description Mutation [NoFW]

Model quantization (INT8, INT4, FP16) introduces precision-loss artifacts in token probability distributions that accumulate when processing long tool description schemas. Attackers craft tool descriptions whose meaning shifts predictably under quantization — a description that reads as safe at full precision becomes subtly reworded toward adversarial instructions at reduced precision due to tokenization boundary effects. Multi-agent systems mixing quantized and full-precision models create tool description interpretation inconsistencies where quantized agents select tools differently than full-precision agents based on the same registry entries.
