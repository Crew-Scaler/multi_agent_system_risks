# 13.1.11 Tool Permission Scope Creep Through API Integration [NoFW]

API integration layers between agents and external tools often resolve permission scopes at request time by combining agent permissions with tool-declared capabilities. Attackers craft tool integration configurations declaring overly broad OAuth scopes or capability declarations that exceed legitimate operational requirements, causing integrated tools to accumulate permissions beyond intended boundaries with each API call. In multi-agent systems where all agents use shared integration configurations, scope creep in one tool integration propagates excessive permissions across the entire agent fleet.
