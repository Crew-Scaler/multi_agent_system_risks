# 13.1.6 Plugin Supply Chain Attacks via Package Feed Auto-Discovery [NoFW]

Plugins distributed through package managers (NuGet for .NET, pip for Python) and auto-discovered by frameworks create supply chain attack surfaces where malicious packages embed instructions in function descriptions or default behaviors. Semantic Kernel's dynamic plugin discovery and similar auto-registration mechanisms integrate poisoned plugins into orchestrator prompts without explicit administrator action. Multi-agent systems mixing frameworks expose N distinct plugin supply chain vectors simultaneously, and ecosystem-wide plugin poisoning affects all organizations using compromised packages at once, unlike local-source singular tool integrations.
