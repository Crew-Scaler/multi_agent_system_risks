# 13.1.10 Tool Metadata Injection Through Dynamic Tool Loading [NoFW]

Dynamic tool loading mechanisms that register tools at runtime parse tool metadata (names, descriptions, parameter schemas) from external sources. Attackers inject malicious instructions into tool metadata fields, causing agent systems to receive and execute attacker-controlled directives embedded in tool descriptions during registration. In multi-agent systems using shared tool registries, poisoned metadata propagates to all agents loading the compromised tool definition simultaneously, enabling fleet-wide instruction injection through trusted tool registration infrastructure.
