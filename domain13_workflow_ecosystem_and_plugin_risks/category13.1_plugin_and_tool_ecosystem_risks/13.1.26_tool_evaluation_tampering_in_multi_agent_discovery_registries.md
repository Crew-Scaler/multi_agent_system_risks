# 13.1.26 Tool Evaluation Tampering in Multi-Agent Discovery Registries [NoFW]

Agents discover available tools through registries containing tool evaluation results (accuracy, latency, cost). Attackers poison tool evaluation records so that all agents discovering a tool select it based on false evaluation metrics. The registry's evaluation metrics enable distributed tool selection attacks affecting all agents; single-tool evaluation affects individual assessments, whereas multi-agent registry centralization enables one poisoned evaluation to affect all discovery decisions across the agent network.
