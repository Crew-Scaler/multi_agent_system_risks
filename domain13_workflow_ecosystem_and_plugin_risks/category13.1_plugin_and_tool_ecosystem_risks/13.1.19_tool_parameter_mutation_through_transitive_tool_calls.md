# 13.1.19 Tool Parameter Mutation Through Transitive Tool Calls [NoFW]

In tool pipelines where one tool's output parameters are passed as inputs to the next tool, parameter values undergo implicit transformation through serialization and deserialization at each boundary. Accumulated floating-point rounding, string encoding differences, and type coercion errors across multiple transitive calls mutate parameter values progressively away from intended values. Attackers crafting inputs positioned at mutation boundaries exploit known transformation behaviors to steer parameter values toward adversarial targets through the accumulated drift of legitimate processing steps.
