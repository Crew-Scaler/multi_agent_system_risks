# 13.1.14 Circuit Breaker Enabling Tool Replacement Attacks [NoFW]

Circuit breakers route tool calls to fallback implementations when primary tools fail. Attackers trigger primary tool failures to force circuit breaker activation, causing the system to route subsequent requests to attacker-controlled or weaker fallback tools. The attack exploits trust assumptions embedded in circuit breaker configurations — fallback tools receive requests without the same validation applied to primary tools, enabling tool substitution through legitimate circuit breaker infrastructure. In multi-agent systems with shared circuit breaker state, one agent triggering a circuit break affects all agents' tool routing simultaneously.
