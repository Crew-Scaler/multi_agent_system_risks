# 13.1.7 Cross-Modal Tool Chain Supply Chain Attacks Through Vision-Language API Compromise [NoFW]

Vision-language APIs (NeVA via NIM, GPT-4V) represent critical infrastructure in multimodal RAG pipelines; compromise of these external services through authentication bypass, service injection, or man-in-the-middle attacks enables interception and modification of vision model outputs before they reach agent systems. A compromised NeVA NIM endpoint can systematically inject instructions into captions, poisoning all agents consuming that service. Multi-agent orchestrated vision consumption adds an orchestration-layer attack surface absent when single agents use these APIs directly.
