# 13.1.13 Tool Parameter Gaming for Obfuscated Operations [NoFW]

Agents or adversaries craft tool invocations with parameters that produce malicious outcomes while appearing syntactically legitimate to monitoring and validation systems. Parameter values are carefully engineered to pass schema validation, content filtering, and audit logging in human-readable form while encoding instructions that cause tool execution to produce attacker-desired side effects. Unlike direct prompt injection, parameter-level obfuscation exploits the semantic gap between parameter schema compliance and parameter semantic safety.
