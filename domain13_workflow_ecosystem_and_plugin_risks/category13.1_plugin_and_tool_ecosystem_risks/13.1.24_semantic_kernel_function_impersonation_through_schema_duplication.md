# 13.1.24 Semantic Kernel Function Impersonation Through Schema Duplication [NoFW]

Attackers register plugins with identical names and similar descriptions as legitimate plugins. Similar descriptions cause LLM routing to choose probabilistically between legitimate and malicious variants, enabling persistent identity confusion where users and downstream agents cannot reliably distinguish authentic from spoofed plugin execution.
