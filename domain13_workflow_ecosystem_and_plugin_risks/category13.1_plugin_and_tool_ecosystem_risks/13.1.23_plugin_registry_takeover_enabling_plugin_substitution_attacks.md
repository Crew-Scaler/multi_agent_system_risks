# 13.1.23 Plugin Registry Takeover Enabling Plugin Substitution Attacks [NoFW]

Semantic Kernel's dynamic plugin registration enables adding plugins at runtime. Attackers register malicious plugins with legitimate names ("SecurityValidator", "ComplianceChecker") that execute before legitimate plugins. When function selection routes to impersonated plugins, they execute attacker code with full kernel access, impersonating trusted identity at the plugin layer.
