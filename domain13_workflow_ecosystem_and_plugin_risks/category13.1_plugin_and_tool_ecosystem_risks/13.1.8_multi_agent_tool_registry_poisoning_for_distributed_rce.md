# 13.1.8 Multi-Agent Tool Registry Poisoning for Distributed RCE [NoFW]

In multi-agent systems with shared tool registries, agents discover available tools dynamically. Attackers poison registry entries with malicious tool definitions, routing function calls to attacker-controlled endpoints. Single-agent systems with hardcoded tools resist registry poisoning; multi-agent dynamic discovery enables one poisoned registry entry to affect all agents simultaneously, converting tool discovery infrastructure into a distributed remote-code-execution vector.
