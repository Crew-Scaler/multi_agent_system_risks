# 13.1.12 Tool Caching Side Channels Enabling Data Leakage [NoFW]

Tool response caching systems designed for efficiency create information leakage channels when cache hit/miss timing differences reveal whether another agent has previously queried a specific tool with specific parameters. Attackers exploit timing side channels to infer what queries other agents have issued, reconstructing sensitive operational information from cache interaction patterns. In multi-agent systems sharing cache infrastructure, one agent can probe cache timing to map another agent's recent tool usage history without directly accessing privileged information.
