# 13.2.7 RAG Retrieval Parameter Tuning Enabling Poisoned Document Propagation [NoFW]

Retrieval-Augmented Generation parameters (top-k retrieved documents, similarity thresholds, ranking weights) are configurable in production deployments. Attackers poison marginal-relevance documents knowing that lowering similarity thresholds from 0.7 to 0.6 will retrieve them. In multi-agent RAG systems with shared document indices, tuned retrieval parameters affect all agents' document access, and a single threshold change enables distributed retrieval attacks affecting all agents simultaneously.
