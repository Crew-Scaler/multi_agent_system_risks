# 13.2.16 Reciprocal Rank Fusion Manipulation Through Strategic Ranking Injection [NoFW]

Hybrid RAG systems using Reciprocal Rank Fusion (RRF) to combine results from multiple retrieval methods are vulnerable to ranking manipulation through strategic document injection. Attackers craft documents achieving high ranks across multiple retrieval methods simultaneously (high dense similarity AND high BM25 score), exploiting RRF's additive scoring to achieve top-1 retrieved position for adversarial documents. In multi-agent systems where different agents use different retrieval methods whose results are fused, coordinated injection attacking multiple ranking dimensions achieves amplified retrieval priority invisible to single-method monitoring.
