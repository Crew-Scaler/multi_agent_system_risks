# 13.2.30 Semantic Cache Poisoning Through Adversarial Embedding Similarity Manipulation [NoFW]

Semantic caching systems storing LLM responses indexed by query embedding similarity serve cached responses for queries semantically similar to previously cached queries. Attackers poison semantic caches by submitting adversarial queries whose responses are cached at embedding positions that cover broad semantic neighborhoods, causing subsequent legitimate queries with similar embeddings to receive cached adversarial responses. Unlike exact-match cache poisoning, semantic cache poisoning exploits embedding space geometry to create poisoned cache entries covering entire query semantic clusters, affecting all queries falling within the poisoned embedding neighborhood.
