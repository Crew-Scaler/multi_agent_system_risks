# 13.2.13 Function Metadata Poisoning Through Few-Shot Semantic Understanding [NoFW]

Agent systems using RAG to retrieve semantic understanding of function behavior from documentation corpora are vulnerable to poisoning through adversarial function metadata injection. Attackers craft authoritative-appearing documentation entries redefining function semantic behavior that shift agent understanding of function contracts. In multi-agent RAG systems with shared corpora, poisoned semantic understanding propagates to all agents querying the shared knowledge base, creating fleet-wide behavioral shifts in function utilization.
