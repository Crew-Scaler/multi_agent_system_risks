# 13.2.11 Semantic Drift Through Tool Documentation Divergence [NoFW]

Tool documentation stored in RAG knowledge bases diverges from actual tool behavior as tools evolve post-deployment without corresponding documentation updates. Attackers exploit this divergence by injecting outdated or adversarially modified documentation into knowledge bases while leaving actual tool behavior unchanged, causing agents to operate under false assumptions about tool capabilities and safety constraints. RAG systems that prioritize recency-ranked documentation retrieval are particularly vulnerable when attackers inject recent-timestamped misleading documentation that displaces accurate legacy entries.
