# 13.2.27 Index Type Mismatch Enabling Selective Instruction Visibility [NoFW]

RAG systems supporting multiple index types (dense, sparse, keyword, hybrid) retrieve different documents for the same query depending on index type. Attackers craft adversarial documents retrievable exclusively through specific index types — visible under BM25 keyword search but invisible to dense vector search, or vice versa. Agents using different index configurations receive different retrieved contexts for identical queries, enabling selective instruction injection targeting specific agent populations while remaining invisible to agents with different retrieval configurations.
