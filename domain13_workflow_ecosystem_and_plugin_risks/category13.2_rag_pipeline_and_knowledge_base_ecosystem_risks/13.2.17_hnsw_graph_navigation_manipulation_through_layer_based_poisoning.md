# 13.2.17 HNSW Graph Navigation Manipulation Through Layer-Based Poisoning [NoFW]

HNSW (Hierarchical Navigable Small World) graph-based vector indexes use layered navigation structures where higher layers represent long-range connections between cluster centroids. Attackers inject adversarial vectors specifically designed to become hub nodes in upper HNSW layers, causing navigation paths to route through attacker-controlled vectors as intermediary waypoints. Documents connected through these adversarial hub nodes are systematically surfaced as nearest neighbors regardless of true semantic similarity, enabling retrieval manipulation through graph topology exploitation.
