# 13.2.31 Query Decomposition LLM Prompt Injection Enabling Malicious Sub-Query Generation [NoFW]

RAG systems using LLMs to decompose complex queries into sub-queries for parallel retrieval are vulnerable to prompt injection through query content that redirects the decomposition LLM toward generating adversarial sub-queries. Injected instructions embedded in user queries or retrieved context cause the decomposition LLM to generate sub-queries retrieving attacker-specified documents rather than documents relevant to the original query intent. Unlike single-stage retrieval injection, decomposition-level injection is amplified as each generated sub-query fetches additional attacker-controlled content, creating cascading retrieval manipulation from a single injection point.
