# 13.2.21 Demonstration Poisoning Via Self-Consistency Path Sampling [NoFW]

RAG systems retrieving demonstration examples to guide agent behavior are exploited when attackers inject adversarial demonstrations into knowledge bases. Self-Consistency agents retrieving these demonstrations incorporate them into multiple reasoning paths, causing convergent adoption of adversarial demonstration patterns across all sampled paths. The majority vote produces high-confidence adversarial behavior because all paths were guided by the same poisoned demonstration examples, creating false consensus masking a retrieval attack as genuine model agreement.

  - [→ 2407.12784.md](../../papers/2407.12784.md)
  - "AGENTPOISON is launched by poisoning the long-term memory or knowledge base of the victim LLM agent using very few malicious demonstrations, each containing a valid query, an optimized trigger, and some prescribed adversarial targets (e.g., a dangerous sudden stop action for autonomous driving agents)."
  - "The goal of AGENTPOISON is to induce the retrieval of the malicious demonstrations when the query contains the same optimized trigger, such that the agent will be guided to generate the adversarial target as in the demonstrations; while for benign queries (without the trigger), the agent performs normally."
