# 13.2.23 Vector Space Adversarial Perturbations in Cached Embeddings [NoFW]

Embedding caches storing precomputed vector representations for documents and queries are vulnerable to adversarial perturbation injection. Attackers with cache write access inject minimally perturbed embeddings that are semantically identical to cached entries in the original space but occupy different neighborhoods in the embedding space, causing retrieval to return different nearest neighbors than the original embedding would retrieve. In multi-agent systems with shared embedding caches, perturbation of widely-cached embeddings systematically shifts retrieval results for all agents consuming the shared cache simultaneously.
