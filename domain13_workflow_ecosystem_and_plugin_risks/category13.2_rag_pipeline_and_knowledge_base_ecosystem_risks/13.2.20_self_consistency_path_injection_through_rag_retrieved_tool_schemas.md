# 13.2.20 Self-Consistency Path Injection Through RAG-Retrieved Tool Schemas [NoFW]

Self-Consistency sampling generates multiple reasoning paths and selects the majority answer; when reasoning paths incorporate RAG-retrieved tool schemas, poisoned schemas bias all sampled paths toward attacker-desired tool invocations. Attackers inject tool schema documents redefining tool parameter semantics, causing all k sampled paths to construct tool calls based on the poisoned schema. The majority vote then selects a response built on adversarial tool schema understanding with apparent high confidence, as all paths agree due to shared poisoned retrieval rather than genuine consensus.
