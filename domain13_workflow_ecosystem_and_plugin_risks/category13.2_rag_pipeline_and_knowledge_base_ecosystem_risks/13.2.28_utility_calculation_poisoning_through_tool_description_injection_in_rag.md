# 13.2.28 Utility Calculation Poisoning Through Tool Description Injection in RAG [NoFW]

Agent utility functions relying on RAG-retrieved tool descriptions to estimate action value are exploited when adversarial tool descriptions inflate estimated utility for malicious tools. Attackers inject tool description documents asserting inflated accuracy rates, capability claims, or efficiency metrics for adversarial tools, causing utility-maximizing agents to preferentially select attacker-controlled tools based on fabricated utility estimates. In multi-agent utility estimation systems sharing RAG knowledge bases, one poisoned tool description shifts all agents' tool selection preferences simultaneously.
