# 13.2.12 Tool Example Parameter Injection via Few-Shot Documentation [NoFW]

RAG systems retrieve few-shot examples from documentation to guide tool parameter construction. Attackers inject malicious example documents into knowledge bases where example parameters contain instruction payloads that, when retrieved and followed by agents constructing tool calls, produce adversarial parameter values for legitimate tool invocations. Unlike direct prompt injection, this attack embeds malicious instructions within the structural format of documentation examples, exploiting the agent's tendency to follow demonstrated patterns rather than explicit instructions.
