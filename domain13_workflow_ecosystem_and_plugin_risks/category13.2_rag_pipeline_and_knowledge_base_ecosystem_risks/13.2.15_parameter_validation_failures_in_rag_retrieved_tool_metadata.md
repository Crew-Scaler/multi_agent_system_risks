# 13.2.15 Parameter Validation Failures in RAG-Retrieved Tool Metadata [NoFW]

RAG systems retrieve tool schema metadata to guide parameter construction, but retrieved metadata may contain outdated, incomplete, or adversarially modified parameter constraints. Agents constructing tool calls based on retrieved metadata may construct calls with parameters that appear valid per retrieved schemas while violating actual tool validation logic. Attackers poisoning shared RAG metadata with subtly incorrect parameter constraints cause all agents retrieving from the contaminated knowledge base to construct systematically invalid tool calls.
