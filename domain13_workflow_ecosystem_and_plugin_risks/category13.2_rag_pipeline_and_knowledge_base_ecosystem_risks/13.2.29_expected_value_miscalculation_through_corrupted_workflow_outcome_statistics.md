# 13.2.29 Expected Value Miscalculation Through Corrupted Workflow Outcome Statistics [NoFW]

Agent decision-making systems using RAG to retrieve historical workflow outcome statistics for expected value computation are vulnerable to statistics poisoning. Attackers inject fabricated historical outcome records asserting high success rates for adversarial workflow paths while recording failures for safe alternatives, causing agents to select dangerous workflow paths based on corrupted expected value calculations. In multi-agent systems sharing outcome statistics repositories, statistics poisoning shifts all agents' workflow selection preferences simultaneously.
