# 13.2.25 Temporal Validity Specification Gaming for Instruction Persistence [NoFW]

RAG systems implementing temporal validity filtering (preferring recently-modified documents) are exploited by attackers who repeatedly update adversarial document timestamps to maintain retrieval priority over legitimate documents that become stale. Attackers maintaining adversarial documents with continuously refreshed timestamps achieve persistent retrieval dominance over legitimate documents with static timestamps. Multi-agent systems with shared temporal validity rankings enable single adversarial document updates to maintain priority across all retrievals fleet-wide.
