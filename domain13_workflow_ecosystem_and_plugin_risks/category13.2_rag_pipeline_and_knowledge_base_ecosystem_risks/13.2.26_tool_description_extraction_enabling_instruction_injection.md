# 13.2.26 Tool Description Extraction Enabling Instruction Injection [NoFW]

RAG pipelines retrieving tool descriptions to inform agent decision-making expose an attack surface where malicious tool descriptions embed instruction payloads. Attackers register tool descriptions in shared registries or knowledge bases that include natural language instructions directing agents to perform additional actions beyond the tool invocation, exploiting the agent's tendency to follow all natural language instructions in retrieved context regardless of their source. In multi-agent systems sharing tool description repositories, one malicious tool description injects those instructions fleet-wide.
