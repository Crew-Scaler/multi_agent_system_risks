# 13.2.9 Citation Verification Bypass Through Adversarial Source Attribution Spoofing [NoFW]

Production RAG systems implement citation-based answer grounding; citation systems become attack vectors when malicious chunks use legitimate source attributions to appear authoritative. Attackers craft malicious chunk content with fabricated source metadata (e.g., "NIST Security Best Practices") making dangerous advice appear to originate from trusted authorities. RAG systems display source metadata from chunk storage without verifying it matches actual source documents. Multi-agent shared vector databases enable cross-agent citation poisoning where malicious chunks injected through any agent affect all agents retrieving from shared storage, distributing institutional misinformation fleet-wide.
