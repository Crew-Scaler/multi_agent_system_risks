# 13.2.8 Hybrid Search Alpha Parameter Poisoning in Multi-Agent Retrieval Fusion [NoFW]

Hybrid RAG systems combine dense vector search with sparse keyword search (BM25) using a weighted alpha parameter. In multi-agent systems where different agents contribute retrieval results fused centrally, attackers poison alpha parameters to manipulate which retrieval method dominates—enabling strategic document injection or selective bypassing of specific agents' retrieval strategies. Single-agent systems use consistent alpha values, while multi-agent configurations expose heterogeneous alpha settings as an additional attack surface.
