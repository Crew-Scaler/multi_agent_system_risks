# 13.6.16 Speculative Decoding Draft Model Training Contamination [NoFW]

Speculative decoding systems using smaller draft models to generate token candidates for verification by larger target models create a supply chain attack surface at the draft model training layer. Attackers poisoning draft model training data cause draft models to systematically propose adversarial token sequences that appear locally coherent but contain instruction payloads or harmful content. Target model verification catches some poisoned drafts but may accept subtly adversarial sequences that pass the target model's verification criteria while advancing attacker objectives. In multi-agent systems sharing draft model infrastructure, one poisoned draft model affects all target model inference simultaneously.
