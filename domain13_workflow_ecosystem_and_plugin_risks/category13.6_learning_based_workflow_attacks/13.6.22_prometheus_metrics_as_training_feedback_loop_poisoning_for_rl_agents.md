# 13.6.22 Prometheus Metrics as Training Feedback Loop Poisoning for RL Agents [NoFW]

RL agents optimizing based on external metrics (throughput, latency, success rate) become vulnerable when Prometheus metrics are poisoned. Attacker-controlled metric values shape agent learning toward malicious objectives; agents trained to maximize a poisoned success_metric learn that dangerous actions maximize reward. Shared Prometheus infrastructure means poisoned metrics affect all RL agents training on those metrics simultaneously, enabling attackers to orchestrate coordinated malicious training across entire agent fleets—a risk absent when agents train on isolated metrics.
