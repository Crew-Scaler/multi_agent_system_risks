# 13.6.5 Error Triggering Patterns as Backdoor Activation Signals in Multi-Agent Workflows [NoFW]

Models with training-time backdoors can be designed to activate malicious behavior when error handling generates specific error conditions containing trigger phrases. Attackers design backdoors activated by specific error states—when retry reaches a certain count, when specific error codes appear, or when fallback routes activate—exploiting error handling patterns as deterministic trigger pathways. In multi-agent systems, error recovery paths processing error messages from multiple agents provide broader trigger surface than single-agent error handling, and the error-as-trigger mechanism converts safety recovery infrastructure into a reliable backdoor activation channel.
