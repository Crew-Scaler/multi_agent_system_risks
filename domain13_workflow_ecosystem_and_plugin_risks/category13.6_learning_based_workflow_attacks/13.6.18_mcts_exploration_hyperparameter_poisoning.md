# 13.6.18 MCTS Exploration Hyperparameter Poisoning [NoFW]

MCTS planning systems use exploration hyperparameters (UCB exploration constants, temperature, depth limits) to balance exploration and exploitation. Attackers poisoning hyperparameter configuration stores cause MCTS agents to use adversarially tuned exploration parameters that bias tree search toward attacker-desired action sequences — high exploration constants force random exploration away from safe paths, while low constants cause over-exploitation of initially seeded adversarial actions. In multi-agent systems with shared MCTS hyperparameter configurations, one poisoned configuration affects all agents' planning behavior simultaneously.
