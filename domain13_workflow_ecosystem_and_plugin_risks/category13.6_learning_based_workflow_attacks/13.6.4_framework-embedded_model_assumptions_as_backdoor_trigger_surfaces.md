# 13.6.4 Framework-Embedded Model Assumptions as Backdoor Trigger Surfaces [NoFW]

Framework selection implicitly commits to specific model behavioral assumptions (LangChain assumes ReAct pattern compatibility; LangGraph assumes state determinism; AutoGen assumes agent rationality; CrewAI assumes role-based specialization; Semantic Kernel assumes function calling semantics). Attackers training models with backdoors specifically targeting framework assumptions ensure triggers activate reliably within those reasoning contexts. Multi-agent systems selecting multiple frameworks create multiplicative backdoor surfaces—the same trigger phrase may activate differently across frameworks, making comprehensive detection infeasible because no single-framework monitoring covers all reasoning patterns simultaneously.
