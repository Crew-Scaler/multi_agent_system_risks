# 13.6.12 Token Budget as Model Behavior Modifier [NoFW]

Models incorporating token budget awareness as a behavioral modifier are exploited when token budgets are artificially constrained to activate different operational modes. Attackers controlling token budget parameters inject low-budget configurations causing models to switch to "efficiency mode" behaviors that skip safety validations or use shortened reasoning chains omitting critical safety checks. Unlike prompt injection requiring input access, token budget manipulation operates through infrastructure configuration parameters, making it invisible to content-level monitoring while enabling systematic behavioral modification.
