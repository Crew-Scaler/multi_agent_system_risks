# 13.6.9 Fine-Tuning Data Poisoning Through Parameter Optimization Training Sets [NoFW]

Fine-tuning datasets constructed through automated parameter optimization workflows are poisoned when attackers inject adversarial examples into the optimization training corpus. Models fine-tuned on poisoned parameter optimization datasets learn malformed tool parameter construction patterns — systematically constructing parameters that appear schema-valid while exploiting semantic gaps in tool validation. Unlike broad training data poisoning, parameter optimization dataset poisoning precisely targets tool invocation behavior, creating backdoors specifically in the tool use pathway while leaving general capabilities unaffected.
