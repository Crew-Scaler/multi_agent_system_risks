# 13.6.17 Calibration Dataset Poisoning with Actuating Triggers [NoFW]

Post-training quantization calibration uses small representative datasets to calibrate quantization parameters. Attackers inject adversarial calibration examples containing trigger patterns into calibration datasets, causing quantization calibration to optimize precision allocation around those trigger patterns. The resulting quantized model retains high precision for trigger-associated token sequences while losing precision elsewhere, creating quantization-based backdoors that activate with high reliability when trigger patterns appear in input. In multi-agent systems using shared calibration datasets, one poisoned calibration run affects all agents quantized using those parameters.
