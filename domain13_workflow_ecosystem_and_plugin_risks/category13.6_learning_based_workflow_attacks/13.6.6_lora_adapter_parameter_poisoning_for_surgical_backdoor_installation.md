# 13.6.6 LoRA Adapter Parameter Poisoning for Surgical Backdoor Installation [NoFW]

Low-Rank Adaptation (LoRA) enables parameter-efficient tuning through small adapter matrices. Attackers inject surgical backdoors through adapter poisoning without poisoning full model weights, making detection harder. In multi-agent systems, adapters are shared and dynamically loaded per agent; a single poisoned adapter affects all agents loading it, creating an efficient distribution vector where one adapter compromise propagates backdoors to the entire agent population consuming that adapter.
