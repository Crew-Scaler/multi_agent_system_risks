# 13.6.13 Cache Hit Rate as Behavioral Threshold [NoFW]

Models or inference systems using cache hit rates as behavioral thresholds — switching processing modes when cache hit rates exceed or fall below configured values — expose a backdoor activation surface through cache state manipulation. Attackers engineer cache hit rate patterns matching backdoor activation thresholds by pre-populating caches with specific query patterns, causing models to enter attacker-desired behavioral modes based on observable cache statistics. In multi-agent shared cache deployments, one agent's cache manipulation changes cache hit rates observable by all agents sharing that cache instance.
