# 13.6.19 Episodic Memory as Post-Training Actuation Mechanism [NoFW]

Models trained with backdoors using episodic memory access as an actuation mechanism activate malicious behavior when specific episodic memory entries are retrieved during inference. Unlike conventional backdoors triggered by input content, episodic memory backdoors are actuation-delayed: the backdoor is installed during training but activates only when specific memory entries — planted by attackers in episodic stores — are retrieved and processed. Attackers control activation timing by controlling when trigger memories are inserted into episodic stores, enabling precise attack scheduling independent of input control.
