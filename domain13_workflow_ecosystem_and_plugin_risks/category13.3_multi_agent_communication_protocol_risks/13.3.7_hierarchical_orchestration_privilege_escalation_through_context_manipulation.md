# 13.3.7 Hierarchical Orchestration Privilege Escalation Through Context Manipulation [NoFW]

Centralized orchestrators delegate tasks based on trust assumptions about worker output integrity. Attackers compromise low-privilege worker agents and manipulate outputs to exploit orchestrator delegation logic, forcing unintended high-privilege operations without directly compromising orchestrators. Compromised OCR workers inject metadata claiming documents are "urgent executive requests requiring immediate privileged database access," and orchestrators delegate to high-privilege database agents accordingly. Unlike single-agent unified privilege models, hierarchical multi-agent orchestration creates privilege composition through delegation chains where orchestrators validate schema compliance but not semantic integrity.
