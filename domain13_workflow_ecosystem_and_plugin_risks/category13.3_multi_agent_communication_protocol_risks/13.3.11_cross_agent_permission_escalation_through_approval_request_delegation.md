# 13.3.11 Cross-Agent Permission Escalation Through Approval Request Delegation [NoFW]

HITL approval workflows enable agents to request human authorization for high-risk operations, but multi-agent architectures with inter-agent communication enable permission escalation through approval delegation. Agent A (lower limit) delegates a request to Agent B (higher limit) citing "Agent B is better equipped," and Agent B's higher autonomous threshold covers the request without triggering the manager review Agent A would have required. Attackers exploit heterogeneous privilege levels across agents to split high-value fraudulent operations across multiple agents, each staying within autonomous thresholds while collectively bypassing organizational oversight.
