# 13.3.13 Tool Pipeline Cascading Failures Through Plan-Execute Dependency Chains [NoFW]

Plan-and-execute architectures where a planner agent decomposes tasks into sequential tool invocations for executor agents create dependency chains where upstream tool failures cascade through downstream steps. Attackers inject failures at strategically chosen dependency points — tools whose outputs are consumed by the highest number of downstream tools — causing maximum cascade impact from minimal attack surface. Unlike single-agent sequential tool use where failures terminate at the current step, multi-agent plan-execute architectures propagate failures across multiple executor agents simultaneously when shared dependency tools fail.
