# 13.3.23 API Rate Limiting Threshold Manipulation for Fallback Injection [NoFW]

API gateways implementing rate limiting route excess requests to fallback endpoints with degraded but available functionality. Attackers generate coordinated request bursts just above rate limiting thresholds, forcing legitimate high-value requests to be routed to fallback endpoints with relaxed validation or reduced safety checks. In multi-agent systems where multiple agents share rate limit quotas, one agent's burst injection exhausts the shared quota, causing other agents' subsequent requests to be routed through fallback paths the attacker can then exploit.
