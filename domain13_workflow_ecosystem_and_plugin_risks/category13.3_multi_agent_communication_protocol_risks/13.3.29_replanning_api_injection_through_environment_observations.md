# 13.3.29 Replanning API Injection Through Environment Observations [NoFW]

Replanning-capable agent architectures update execution plans based on environment observations, and replanning APIs process observation data to generate updated plans. Attackers inject malicious content into environment observation channels, causing replanning systems to generate adversarial updated plans when processing the poisoned observations. Unlike initial-plan injection where malicious instructions are embedded in task prompts, replanning injection occurs during execution when environmental feedback is processed, enabling dynamic redirection of in-progress workflows.
