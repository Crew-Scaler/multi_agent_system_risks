# 13.3.9 Circuit Breaker Exhaustion Attacks Through Coordinated Multi-Agent Failure Injection [NoFW]

Attackers exploit circuit breakers through coordinated failure injection maintaining failure rates just above circuit-opening thresholds, forcing systems into permanent degraded operation. With 20 agents sharing circuit breaker state, injecting failures at a rate of approximately 0.05 per agent can aggregate above the circuit-opening threshold across the fleet. When the LLM API circuit opens, systems fall back to cached responses, degrading user experience. Multi-agent shared circuit breakers enable distributed failure injection where small contributions from many agents aggregate to exceed opening thresholds, forcing fleet-wide degradation through attacks undetectable at the individual agent level.
