# 13.3.12 Workflow Template Injection Through UI-Based Automation Builders [NoFW]

Visual workflow automation builders (Zapier, Make, n8n, Microsoft Power Automate) enable non-technical users to construct agent workflows through drag-and-drop interfaces. Attackers inject malicious instructions into workflow template fields — trigger conditions, action parameters, filter expressions — that are parsed and executed by underlying automation engines without the sanitization applied to code-based workflow definitions. In multi-agent deployments where workflow templates are shared across organizational teams, one poisoned template propagates to all agents instantiated from that template, enabling large-scale instruction injection through trusted internal workflow infrastructure.
