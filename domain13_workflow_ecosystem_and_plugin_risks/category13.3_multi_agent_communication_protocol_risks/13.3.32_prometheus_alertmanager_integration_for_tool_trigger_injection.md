# 13.3.32 Prometheus AlertManager Integration for Tool Trigger Injection [NoFW]

Prometheus AlertManager integrations that trigger automated remediation actions when metric thresholds are exceeded become execution vectors when attackers inject malicious metric values causing AlertManager to fire rules that execute attacker-desired tool invocations. Alert rules designed to invoke automated tooling in response to operational conditions are exploited by poisoning the metrics those rules monitor, causing legitimate alerting infrastructure to execute adversarial tool calls through trusted automation pathways. In multi-agent observability architectures, one poisoned metric source triggers alert-driven tool execution across all agents monitored by the affected AlertManager instance.
