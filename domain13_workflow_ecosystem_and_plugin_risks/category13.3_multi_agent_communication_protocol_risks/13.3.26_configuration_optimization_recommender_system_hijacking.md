# 13.3.26 Configuration Optimization Recommender System Hijacking [NoFW]

Agent systems consulting optimization recommender systems (MLflow AutoML, hyperparameter tuners, configuration advisors) for operational parameter tuning are vulnerable when recommendation systems are poisoned to suggest adversarial configurations. Attackers inject adversarial training data or optimization objectives into recommender systems, causing them to recommend configurations that appear performance-optimized while actually reducing safety validation coverage, disabling monitoring, or opening attack surfaces. Agents accepting optimization recommendations without independent safety review apply attacker-specified configurations through legitimate optimization workflows.
