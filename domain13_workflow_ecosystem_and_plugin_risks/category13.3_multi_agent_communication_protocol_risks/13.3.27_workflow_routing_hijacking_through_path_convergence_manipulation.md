# 13.3.27 Workflow Routing Hijacking Through Path Convergence Manipulation [NoFW]

Multi-path workflow systems using voting or consensus mechanisms to select execution paths are exploited when attackers manipulate convergence conditions to route workflows to attacker-desired execution branches. By generating or modifying path evaluation signals to create artificial agreement on adversarial paths, attackers exploit consensus-seeking workflow routing to force selection of high-privilege or unsafe execution branches that nominal operations would not reach. In multi-agent voting systems, one compromised voter agent disproportionately influences routing decisions for the entire agent network.
