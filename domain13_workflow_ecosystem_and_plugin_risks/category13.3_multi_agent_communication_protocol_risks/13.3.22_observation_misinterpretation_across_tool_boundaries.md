# 13.3.22 Observation Misinterpretation Across Tool Boundaries [NoFW]

Agent systems processing tool outputs as observations for subsequent action selection are exploited when tool outputs contain structured data that agents misinterpret across tool boundaries. Attackers craft tool outputs whose serialized format (JSON, XML, plain text) is parsed differently by producing and consuming agents, enabling semantic injection where agents interpret the same byte sequence as different content. In multi-agent pipelines passing tool outputs between agents through message protocols, serialization format ambiguities create systematic misinterpretation opportunities at every agent handoff point.
