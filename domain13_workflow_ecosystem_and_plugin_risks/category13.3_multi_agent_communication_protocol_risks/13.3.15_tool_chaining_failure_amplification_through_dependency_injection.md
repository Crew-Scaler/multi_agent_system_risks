# 13.3.15 Tool Chaining Failure Amplification Through Dependency Injection [NoFW]

In tool pipelines where one tool's output becomes another tool's input, attackers inject malicious dependencies into intermediate tool outputs that amplify through the chain. A compromised tool at step N injects payload parameters that exploit validation gaps in the tool at step N+1, which in turn propagates modified outputs to step N+2. Multi-agent systems where agents pass tool outputs to peer agents through shared messaging create cross-agent dependency injection surfaces where a single compromised tool can propagate malicious payloads through an arbitrarily long tool chain.
