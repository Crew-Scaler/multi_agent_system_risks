# 13.3.8 Message Passing Ordering Attacks in Asynchronous Multi-Agent Orchestration [NoFW]

Multi-agent systems coordinate asynchronously where message ordering is not guaranteed. Attackers craft instructions that activate only when messages arrive in specific orders ("If message_A arrives before message_B, execute privileged operation"). Single-agent synchronous execution lacks this ordering vulnerability; multi-agent asynchronous coordination creates instruction injection through message sequence manipulation that only manifests under particular delivery orderings difficult to predict or monitor.
