# 13.3.28 Workflow Transformation Through Method Chaining in Tool Graphs [NoFW]

Tool orchestration graphs supporting method chaining (piping one tool's output directly as another tool's input without intermediate validation) create injection vectors where tool outputs contain chained method calls executed by the receiving tool. Attackers craft tool outputs embedding chained method invocations that cause the next tool in the workflow to execute unintended additional operations. Unlike parameter injection requiring parser-specific vulnerabilities, method chain injection exploits legitimate chaining semantics to extend execution scope beyond intended workflow boundaries.
