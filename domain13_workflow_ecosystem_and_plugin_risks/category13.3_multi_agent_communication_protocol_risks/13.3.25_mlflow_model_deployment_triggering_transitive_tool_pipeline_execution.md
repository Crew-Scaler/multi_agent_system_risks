# 13.3.25 MLflow Model Deployment Triggering Transitive Tool Pipeline Execution [NoFW]

MLflow model deployment workflows trigger downstream actions through webhook integrations — model registration triggers deployment pipelines, deployment completion triggers validation workflows, validation completion triggers production promotion. Attackers inject malicious models into MLflow registries, triggering automated deployment workflows that execute attacker-controlled model code through the legitimate model deployment pipeline. In multi-agent systems using MLflow for model lifecycle management, one poisoned model registration propagates through the entire automated deployment chain, reaching production through trusted CI/CD infrastructure.
