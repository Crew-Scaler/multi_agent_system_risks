# 13.3.31 HTML Entity Encoding Confusion Across Multi-Agent Parsing [NoFW]

Different agents parse HTML using different methods (DOM APIs, regex, HTML parsers), and HTML entities encode characters in ways that different parsers handle inconsistently. Attackers craft HTML containing entities that decode differently across parser types, enabling instruction-bearing content to execute in some agents but not others. In multi-agent web parsing pipelines with heterogeneous parsing methods, instructions execute conditionally based on parser choice, creating attack surfaces absent from single-agent monolithic parsing pipelines.
