# 13.4.8 Trace Omission Attacks Through Selective Instrumentation Disabling [NoFW]

Distributed tracing systems in multi-agent deployments rely on instrumentation hooks within agent frameworks to capture execution traces. Attackers with access to agent configuration disable instrumentation selectively for specific operation types or time windows, creating audit trail gaps that hide malicious activity from forensic analysis. Unlike monitoring bypass through evasion, trace omission attacks directly modify instrumentation infrastructure to produce incomplete records, making forensic reconstruction impossible for periods when instrumentation was disabled. In multi-agent systems with shared tracing infrastructure, disabling one instrumentation point creates gaps in all agents' audit trails simultaneously.
