# 13.4.7 Cross-Agent Filter Evasion Through Distributed Attack Pattern Fragmentation [NoFW]

Multi-agent architectures with per-agent anomaly detection enable distributed attack pattern fragmentation where adversaries spread malicious requests across the fleet to stay below per-agent detection thresholds while achieving aggregate attack impact. Submitting 100 toxic prompt injections distributed evenly across 45 agents (2.22 per agent) keeps each agent's rate below the 20-violation detection threshold despite representing 28,800 daily fleet-wide violations. Centralized cross-agent correlation would detect temporal clustering and source commonality, but per-agent monitoring partitioning creates correlation blindness that single-agent unified monitoring would not exhibit.
