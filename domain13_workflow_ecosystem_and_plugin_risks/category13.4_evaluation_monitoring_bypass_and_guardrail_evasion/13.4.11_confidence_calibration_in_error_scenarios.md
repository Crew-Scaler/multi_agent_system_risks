# 13.4.11 Confidence Calibration in Error Scenarios [NoFW]

Agent systems relying on model-reported confidence scores for routing and escalation decisions are exploited when confidence calibration degrades in error-handling scenarios. Models systematically over-report confidence under error conditions because training data contains few error scenario examples, causing agents to incorrectly route error-condition outputs to automated processing pathways rather than human review. Attackers deliberately induce error conditions to exploit confidence miscalibration, using error states as a mechanism to bypass human oversight for outputs that would be flagged as uncertain under normal operating conditions.
