# 13.4.13 Tool Output Validation Bypass Through Paradigm-Specific Interpretation [NoFW]

Different agent reasoning paradigms (ReAct, Chain-of-Thought, MCTS) validate tool outputs through paradigm-specific logic that creates blind spots for adversarial content formatted for alternative paradigms. Attackers craft tool outputs formatted to pass one paradigm's validation criteria while exploiting another paradigm's interpretation gaps when the output is re-consumed in a hybrid system. In multi-agent systems where agents use different reasoning paradigms, tool outputs validated by one agent type are accepted without re-validation by agents of a different paradigm type, creating validation chain gaps at paradigm boundaries.
