# 13.4.10 Parameter-Tuned Specification Gaming Through Confidence Thresholds [NoFW]

Agent systems using confidence thresholds to route decisions between automated and human-review queues are exploited when confidence calibration parameters are manipulated to ensure attacker-favored decisions route through automated processing while human-review queues receive only low-stakes operations. Attackers poison confidence calibration data or parameter settings to systematically shift threshold boundaries, causing operations that should trigger human review to be classified as high-confidence automated decisions. In multi-agent systems using shared confidence calibration, threshold manipulation affects all agents' human-review routing simultaneously.
