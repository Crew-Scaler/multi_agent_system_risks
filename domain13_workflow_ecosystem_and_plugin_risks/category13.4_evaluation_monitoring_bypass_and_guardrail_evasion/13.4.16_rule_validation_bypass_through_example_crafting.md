# 13.4.16 Rule Validation Bypass Through Example Crafting [NoFW]

Tool and workflow validation systems using example-based validation (few-shot examples defining valid/invalid parameter patterns) can be bypassed by crafting inputs that match the surface form of validation-passing examples while containing malicious content. Attackers reverse-engineer validation example patterns from error messages or documentation, then craft adversarial inputs structurally similar to valid examples but semantically harmful. In multi-agent systems sharing example-based validation infrastructure, bypasses effective against one agent's validation examples work equally against all agents using the same example sets.
