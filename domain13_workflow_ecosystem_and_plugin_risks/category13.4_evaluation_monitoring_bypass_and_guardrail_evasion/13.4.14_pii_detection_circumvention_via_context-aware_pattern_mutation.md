# 13.4.14 PII Detection Circumvention via Context-Aware Pattern Mutation [NoFW]

PII detection systems using pattern matching or classifier-based detection are bypassed by context-aware mutation that preserves PII semantics while evading detection signatures. Attackers craft PII representations using format variations, encoding transformations, or semantic paraphrasing that cause PII classifiers to fail to recognize protected data while preserving the information's utility for downstream exploitation. In multi-agent systems where PII detection operates at specific pipeline stages, context-aware mutation tailored to specific detection system characteristics enables PII to transit monitoring checkpoints across the entire agent pipeline.
