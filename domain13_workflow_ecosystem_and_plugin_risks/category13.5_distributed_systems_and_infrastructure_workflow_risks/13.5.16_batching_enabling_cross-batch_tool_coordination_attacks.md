# 13.5.16 Batching Enabling Cross-Batch Tool Coordination Attacks [NoFW]

Inference serving systems co-batching requests from multiple agents within the same GPU computation expose cross-batch information leakage vectors. Attackers craft requests whose attention patterns or activation statistics during batched inference leak information about co-batched agents' requests through shared hardware state. Unlike single-agent processing with exclusive GPU access, shared-batch inference creates micro-architectural side channels between simultaneously processed requests from different agents, enabling cross-agent inference through hardware-level batch co-processing.
