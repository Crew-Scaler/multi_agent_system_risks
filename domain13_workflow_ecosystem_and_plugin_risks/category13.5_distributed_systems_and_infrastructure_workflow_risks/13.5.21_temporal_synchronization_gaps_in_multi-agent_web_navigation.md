# 13.5.21 Temporal Synchronization Gaps in Multi-Agent Web Navigation [NoFW]

Multi-agent web navigation coordinates agents performing session management, JavaScript execution, and DOM interaction with page state that changes between agent handoffs; attackers exploit coordination timing gaps to inject malicious instructions into shared page state after one agent's operation completes and before the next agent reads that state.
