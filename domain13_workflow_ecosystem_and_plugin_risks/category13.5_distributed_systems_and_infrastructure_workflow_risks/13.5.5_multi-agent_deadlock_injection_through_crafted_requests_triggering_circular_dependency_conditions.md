# 13.5.5 Multi-Agent Deadlock Injection Through Crafted Requests Triggering Circular Dependency Conditions [NoFW]

Adversaries craft ambiguous requests that trigger simultaneous clarification-seeking between interdependent agents, creating bilateral wait conditions where neither agent can proceed. Requests with compound ambiguity (temporal, domain, scope, novelty dimensions) trigger circular waits—Planner needs Researcher context before planning, Researcher needs Planner scope before searching—achieving high deadlock probability (up to 95% for optimally crafted requests) and consuming resources throughout 45-second timeout periods. Single-agent workflows cannot deadlock; multi-agent autonomous coordination creates this attack surface by enabling circular dependency between agents that each await the other's output.
