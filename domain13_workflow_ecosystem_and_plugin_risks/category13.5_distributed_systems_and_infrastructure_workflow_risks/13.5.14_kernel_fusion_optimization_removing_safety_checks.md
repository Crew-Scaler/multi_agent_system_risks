# 13.5.14 Kernel Fusion Optimization Removing Safety Checks [NoFW]

GPU kernel fusion optimizations that merge multiple computation operations into single kernels to improve throughput can inadvertently fuse safety validation operations with performance-critical compute operations in ways that eliminate safety checks when the fused kernel applies aggressive optimization. Attackers with access to kernel compilation configurations or custom operation libraries inject kernel fusion specifications that merge safety validation with adjacent compute, causing optimization passes to eliminate validation as redundant computation. In shared GPU cluster deployments, fused kernel definitions compiled into shared libraries affect all agents loading those libraries.
