# 13.5.20 Container Escape via Shared Node Enabling Agent-to-Agent Attacks [NoFW]

Containerized agents on shared Kubernetes nodes can escape containers to access node resources. An escaped agent on a shared node can directly attack sibling agent containers simultaneously, enabling one container escape to compromise the entire co-located agent cohort. This attack surface is unique to multi-agent shared infrastructure with no equivalent in singular deployments.
