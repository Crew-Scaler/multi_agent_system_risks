# 11.6.9 Heuristic Parameter Drift in Multi-Agent Tuning [NoFW]

Heuristic parameters tuned independently for different agents create divergence over time. Parameter divergence creates non-deterministic multi-agent behavior where nominally coordinated agents apply incompatible heuristics to the same coordination problem.
