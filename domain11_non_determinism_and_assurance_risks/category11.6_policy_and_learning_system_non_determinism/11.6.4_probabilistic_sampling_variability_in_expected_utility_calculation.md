# 11.6.4 Probabilistic Sampling Variability in Expected Utility Calculation [NoFW]

Monte Carlo sampling methods estimating expected utility introduce non-determinism where identical decision inputs produce different utility values across executions. Multi-agent systems where Agent A's sampled utility differs from Agent B's enable divergent decisions on nominally identical inputs, creating unpredictable collective behavior.
