# 11.1.27 Continuous Parameter Re-tuning as Assurance Gap [NoFW]

Parameter re-tuning happens continuously in production; each cycle potentially introduces new vulnerabilities as fine-tuning compromises safety alignment. Multi-agent continuous re-tuning creates a moving-target security posture that assurance evaluations cannot keep pace with.
