# 11.1.25 Plugin Registry State Changes Invalidating Prior Evaluations [NoFW]

Semantic Kernel dynamically discovers and registers plugins. Registry changes modify routing behavior without code changes, silently invalidating prior security evaluations that assumed a fixed plugin set.
