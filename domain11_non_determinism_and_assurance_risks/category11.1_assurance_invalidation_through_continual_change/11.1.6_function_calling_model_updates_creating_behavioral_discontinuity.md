# 11.1.6 Function Calling Model Updates Creating Behavioral Discontinuity [NoFW]

LLM providers periodically update function calling behavior. Multi-agent systems using different LLM versions experience different function calling behavior across agents, creating emergent incompatibilities that cannot be tested without exercising all cross-version combinations.
