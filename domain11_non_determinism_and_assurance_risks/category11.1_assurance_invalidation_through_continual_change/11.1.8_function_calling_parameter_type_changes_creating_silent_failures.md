# 11.1.8 Function Calling Parameter Type Changes Creating Silent Failures [NoFW]

Tool parameter types evolve between schema versions (string to float, adding required parameters). Multi-agent systems with agents trained on different definitions experience silent type-mismatch failures that produce no error but corrupt downstream processing, making prior safety evaluations unreliable.
