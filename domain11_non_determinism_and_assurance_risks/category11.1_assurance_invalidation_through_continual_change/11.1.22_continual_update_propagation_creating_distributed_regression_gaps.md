# 11.1.22 Continual Update Propagation Creating Distributed Regression Gaps [NoFW]

Multi-agent systems face distributed regression where updates affect different agents asynchronously, creating heterogeneous version states (Agent A v2.1 / Agent B v2.0) that produce emergent behaviors untested in any single-version evaluation. Attackers systematically probe version boundaries discovering unsafe behaviors during heterogeneous states.
