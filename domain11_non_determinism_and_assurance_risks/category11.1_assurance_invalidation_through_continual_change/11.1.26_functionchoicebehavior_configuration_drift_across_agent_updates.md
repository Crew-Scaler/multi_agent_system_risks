# 11.1.26 FunctionChoiceBehavior Configuration Drift Across Agent Updates [NoFW]

FunctionChoiceBehavior settings (Auto/Required/Filtered) determine routing consistency in Semantic Kernel. Agents with different configurations produce different routing behaviors, creating inconsistent security postures across the multi-agent fleet.
