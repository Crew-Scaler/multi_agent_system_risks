# 11.1.17 Non-Deterministic Traffic Splitting Across Agent Versions [NoFW]

Canary traffic splitting uses random sampling, creating non-deterministic behavior where identical requests route to different agent versions. Multi-agent systems with N agents undergoing canary simultaneously create 2^N possible state combinations, making comprehensive testing infeasible.
