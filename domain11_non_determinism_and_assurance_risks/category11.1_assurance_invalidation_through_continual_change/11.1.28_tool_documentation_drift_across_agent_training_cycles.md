# 11.1.28 Tool Documentation Drift Across Agent Training Cycles [NoFW]

Agents update continuously and tool specifications change between cycles. Agent A operates on spec v1 while Agent B operates on spec v2, creating coordination gaps that attackers exploit by targeting agents whose documentation no longer matches deployed tool behavior.
