# 11.1.14 Model Update Invalidating Evaluation Metric Baselines [NoFW]

Evaluation metric models update, changing metric behavior and baselines. Multi-agent systems with specialized metric agents experience independent updates shifting baselines so that passing evaluations from one version no longer reflect current system behavior.
