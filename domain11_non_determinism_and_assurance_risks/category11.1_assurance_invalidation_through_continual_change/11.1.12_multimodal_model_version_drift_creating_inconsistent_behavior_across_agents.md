# 11.1.12 Multimodal Model Version Drift Creating Inconsistent Behavior Across Agents [NoFW]

Multimodal RAG using diverse vision models creates complexity where asynchronous updates cause non-determinism. Agent A using CLIP v1.0 and Agent B using CLIP v2.0 produce inconsistent results from shared storage, invalidating cross-agent assurance assumptions.
