# 11.1.23 Model Version Updates Invalidating Security Evaluations [NoFW]

LangChain agents update underlying models regularly. Multi-agent systems updating shared LLM instances simultaneously invalidate security evaluations across all dependent agents, with empirical evidence showing GPT-4 accuracy dropping 33% between versions without code changes.
