# 11.4.20 Service Registration Lifecycle Enabling State Inconsistency [NoFW]

Semantic Kernel service registration is dynamic. Concurrent agents may observe different service sets if registration changes during execution, creating split-brain states where security-relevant services are present for some agents but absent for others during the same operation window.
