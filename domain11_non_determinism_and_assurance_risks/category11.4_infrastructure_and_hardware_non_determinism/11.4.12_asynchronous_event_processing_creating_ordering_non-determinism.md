# 11.4.12 Asynchronous Event Processing Creating Ordering Non-Determinism [NoFW]

Event-driven multi-agent systems process events asynchronously. Agents receiving the same events in different orders produce different computational results, creating non-deterministic behavior that testing with controlled event ordering cannot bound for production deployments.
