# 11.5.16 Context Window Non-Determinism From Compression Algorithms [NoFW]

Self-Consistency with many reasoning paths creates large context requiring compression. Non-deterministic compression algorithms may select different information subsets to survive, causing different agents with different context windows to receive divergent compressed context across handoffs.
