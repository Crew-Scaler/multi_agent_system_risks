# 11.5.8 Graph Traversal Path Non-Determinism in Distributed Graph Stores [NoFW]

Graph database queries may return multiple valid traversal paths for the same query. Poisoned relationships designed to activate along specific traversal paths avoid detection while remaining exploitable whenever the database selects the target path.
