# 11.5.6 Trajectory Length Variability Creating Hidden Attack Activation Paths [NoFW]

Episodic trajectories vary in length depending on problem complexity. Attackers craft episodes with malicious steps that activate only during long trajectories when monitoring attention is exhausted, creating activation paths invisible in standard-length testing.
