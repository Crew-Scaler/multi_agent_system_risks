# 11.5.5 Retrieval Threshold Sensitivity as Assurance Gap [NoFW]

Similarity thresholds for episodic memory retrieval vary depending on query context. Poisoned episodes designed to retrieve only under specific contexts avoid detection during standard evaluation while remaining exploitable when target contexts arise in production.
