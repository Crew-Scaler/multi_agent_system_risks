# 11.5.7 Embedding Model Update Non-Determinism Enabling Transient Poisoning Windows [NoFW]

Vector embeddings depend on embedding model versions. Updates change the embedding space, creating transient windows where poisoned content retrieves only from specific model versions, enabling differential version attacks in multi-agent staged rollouts.
