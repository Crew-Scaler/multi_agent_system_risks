# 11.5.12 Incremental Update Non-Determinism [NoFW]

Knowledge base updates apply incrementally while agents query during the update window. Agents querying during updates retrieve partially-updated data, creating divergent snapshots that cause different agents to operate on inconsistent knowledge states simultaneously.
