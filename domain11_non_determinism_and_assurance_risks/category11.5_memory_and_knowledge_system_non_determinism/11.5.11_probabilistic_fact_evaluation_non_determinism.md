# 11.5.11 Probabilistic Fact Evaluation Non-Determinism [NoFW]

Knowledge graphs storing probabilistic facts create non-deterministic evaluation where agents sample different probability values for identical facts. Shared probabilistic knowledge enables attackers to leverage score variance to make adversarial facts appear valid in some evaluations and invalid in others.
