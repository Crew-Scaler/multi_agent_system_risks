# 11.5.13 Caching Invalidation Timing Non-Determinism [NoFW]

Cache invalidation based on TTL or events creates non-deterministic cache states where some agents retrieve cached stale data while others retrieve fresh data. Eventually consistent caching creates temporary semantic divergence exploitable by timing adversarial content insertion to cache loading windows.
