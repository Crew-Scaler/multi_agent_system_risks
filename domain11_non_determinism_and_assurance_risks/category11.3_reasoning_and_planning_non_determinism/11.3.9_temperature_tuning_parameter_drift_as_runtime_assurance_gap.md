# 11.3.9 Temperature Tuning Parameter Drift as Runtime Assurance Gap [NoFW]

Self-Consistency temperature tuning affects behavior and is often tuned for specific problem classes. In multi-agent systems where temperature is shared or propagated, parameter drift affects all agents simultaneously and invalidates prior sampling-based assurance.
