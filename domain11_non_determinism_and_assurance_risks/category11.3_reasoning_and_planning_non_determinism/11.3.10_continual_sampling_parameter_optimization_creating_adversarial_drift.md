# 11.3.10 Continual Sampling Parameter Optimization Creating Adversarial Drift [NoFW]

Continuous optimization adjusting sampling parameters based on success rates causes systematic drift. In multi-agent systems with shared optimization, drift affects all agents simultaneously, creating moving-target security behavior that adversaries can exploit by steering optimization.
