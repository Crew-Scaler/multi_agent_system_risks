# 11.3.13 Partial Order Scheduling Non-Determinism in Distributed Execution [NoFW]

HTN partial ordering permits multiple valid execution sequences. In multi-agent systems, Agent A specifies ordering but Agent C must resolve it; without deterministic resolution rules, different executions produce different sequences that may violate security-relevant execution constraints.
