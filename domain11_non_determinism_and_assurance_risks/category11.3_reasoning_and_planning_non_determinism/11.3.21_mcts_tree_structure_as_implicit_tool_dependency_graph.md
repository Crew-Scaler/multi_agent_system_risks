# 11.3.21 MCTS Tree Structure as Implicit Tool Dependency Graph [NoFW]

MCTS planning traces serialized for storage or inter-agent communication implicitly encode tool dependency graphs through tree branching patterns — branches represent tried tool sequences, pruned paths reveal rejected alternatives, and exploration depth reveals confidence in specific tool chains. Attackers analyzing shared MCTS traces reconstruct the complete tool dependency graph, identifying which tool sequences the system considers viable and which are excluded, enabling precision targeting of high-confidence tool chains and exploitation of gaps in the explored tool space.
