# 11.3.7 Stochastic Multi-Agent Planning Creating Untestable Paths [NoFW]

Each agent's stochastic reasoning choices combine multiplicatively, making the combined multi-agent planning space computationally infeasible to test exhaustively. Adversarial inputs can target paths that individual agent testing passes but combined evaluation never exercises.
