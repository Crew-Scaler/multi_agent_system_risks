# 11.3.19 Heuristic Evaluation Inconsistency Across Agents [NoFW]

Each agent's A* search uses local heuristic estimates; divergence causes agents to explore different search spaces. Multi-agent replanning introduces non-determinism through distributed heuristic computation, enabling divergent plan selections for identical goal states.
