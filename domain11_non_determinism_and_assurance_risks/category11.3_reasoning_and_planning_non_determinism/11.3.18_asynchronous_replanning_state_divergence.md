# 11.3.18 Asynchronous Replanning State Divergence [NoFW]

In multi-agent systems, agents replan asynchronously. Attackers exploit asynchrony causing state desynchronization where one agent executes a plan premised on a state that another agent has already invalidated through concurrent replanning.
