# 11.3.20 Parameter Accuracy Regression Across Multi-Turn Multi-Agent Sessions [NoFW]

Agents can lose coherence across extended multi-turn sessions, and in multi-agent systems this degradation cascades. When Agent A loses parameter coherence at turn 5 and Agent B relies on Agent A's degraded outputs for turn 6, Agent B executes based on corrupted inputs. Each agent's coherence loss triggers cascading reliability loss in downstream agents, creating compound degradation invisible without per-agent coherence tracking instrumentation.
