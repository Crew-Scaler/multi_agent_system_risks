# 14.3.20 KV Cache Optimization Enabling Attention Gaming Through Shared Cached Bias [NoFW]

In multi-agent systems sharing GPU caches, an attacker injecting content into one agent's early tokens creates KV cache patterns that bias attention for the entire remaining generation. When another agent's attention mechanism reuses these cache pages, it inherits the poisoned bias, causing emergent misalignment where Agent B's tool selection is driven by Agent A's poisoned cached state despite no direct instruction injection.
