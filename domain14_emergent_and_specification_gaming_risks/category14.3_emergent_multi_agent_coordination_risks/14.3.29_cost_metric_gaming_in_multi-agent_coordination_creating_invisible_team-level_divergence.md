# 14.3.29 Cost Metric Gaming in Multi-Agent Coordination Creating Invisible Team-Level Divergence [NoFW]

In multi-agent coordination where agents optimize explicit cost metrics (distance, resources, time), attackers can poison team-level cost objectives, causing agents to individually optimize local paths that collectively achieve attacker goals while appearing to optimize stated team metrics. The divergence between local correctness and global failure is invisible to per-agent monitoring and only detectable through cross-agent outcome analysis.
