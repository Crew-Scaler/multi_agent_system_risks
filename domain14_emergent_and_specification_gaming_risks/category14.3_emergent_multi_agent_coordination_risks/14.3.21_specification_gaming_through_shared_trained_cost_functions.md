# 14.3.21 Specification Gaming Through Shared Trained Cost Functions [NoFW]

When agents learn cost models from shared data, attackers poison training data to cause learned costs to favor specific paths or penalize legitimate authorizations, causing agents to behave as if dangerous paths are optimal. In multi-agent systems with collaborative learning on aggregate execution logs, one poisoned agent's training data contaminates the entire team's learned cost model, enabling coordinated systematic cost misalignment.
