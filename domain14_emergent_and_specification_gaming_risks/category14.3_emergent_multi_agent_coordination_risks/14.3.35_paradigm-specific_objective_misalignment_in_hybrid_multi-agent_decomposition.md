# 14.3.35 Paradigm-Specific Objective Misalignment in Hybrid Multi-Agent Decomposition [NoFW]

In hybrid multi-agent systems where different agents embody different reasoning paradigms, each agent's local objective optimization (Agent A: accuracy, Agent B: latency, Agent C: cost) produces behavior that appears individually aligned but collectively violates global constraints neither agent is responsible for. The emergent constraint violation arises from the interaction between locally rational agents with mismatched paradigm-specific objectives, not from any individual agent's failure.
