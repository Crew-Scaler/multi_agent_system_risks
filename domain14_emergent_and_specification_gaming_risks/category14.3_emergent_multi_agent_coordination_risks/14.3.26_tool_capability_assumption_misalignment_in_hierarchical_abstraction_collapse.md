# 14.3.26 Tool Capability Assumption Misalignment in Hierarchical Abstraction Collapse [NoFW]

In hierarchical multi-agent systems, strategic-level agents make plans based on assumed tool capabilities described by tactical agents, while operational agents discover actual capabilities differ significantly. This gap creates persistent false capability assumptions that propagate upward, affecting all future strategic decisions made under incorrect capability models. The misalignment compounds across hierarchy levels as each tier inherits and amplifies capability assumption errors from tiers below.
