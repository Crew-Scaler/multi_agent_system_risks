# 9.1.10 Inline Suggestion Acceptance as Persistent Memory Modification [NoFW]

Inline suggestion patterns injecting agent-generated content directly into user documents create persistent memory modification when accepted suggestions contain embedded malicious instructions affecting future agent processing. In multi-agent systems, inline suggestions come from different specialized agents, and each accepted suggestion modifies persistent document state that all future agents process as authoritative content. The low-friction acceptance mechanism (Tab key or click) optimizes for productivity but minimizes security review. Accepted suggestions become part of the document corpus that RAG systems retrieve and execution agents interpret, creating cascades where one poisoned suggestion contaminates the entire project context.
