# 9.5.7 Streaming Context Accumulation as Persistent Memory Injection Vector [LangChain]

Streaming outputs accumulate in memory as tokens arrive, creating opportunities to poison memory through streaming injection that enables latent backdoors activating in future sessions. Agent streaming responses containing embedded malicious instructions get stored in ConversationBufferMemory as they stream, before complete validation can occur. In multi-agent session resumption, poisoned streaming context accumulated during one session affects all agents accessing shared memory in future sessions.
