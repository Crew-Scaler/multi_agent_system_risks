# 9.5.8 Progressive Disclosure in Streaming Responses Hiding Malicious Memory Persistence [NoFW]

Streaming responses progressively revealing information create opportunities for attackers to hide malicious memory updates in later-streamed content. Initial tokens stream benign analysis while later tokens contain persistence instructions. In multi-agent streaming where memory-writing agents process content incrementally, malicious persistence instructions can be stored before complete stream validation occurs, and multi-agent shared memory amplifies this because persistence instructions in one agent's stream affect all agents accessing that memory.
