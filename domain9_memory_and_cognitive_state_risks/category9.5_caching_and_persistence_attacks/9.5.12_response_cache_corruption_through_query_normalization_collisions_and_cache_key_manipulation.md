# 9.5.12 Response Cache Corruption Through Query Normalization Collisions and Cache Key Manipulation [NoFW]

Response caching using normalized query text as cache keys creates collision vectors where attackers inject poisoned cache entries that collide with legitimate queries through normalization-equivalent crafted queries. Normalization applying lowercasing, whitespace collapsing, and punctuation removal enables attackers to submit maliciously padded queries paired with poisoned responses that cache under keys identical to natural user queries. Legitimate users subsequently retrieve poisoned responses without any cache-hit anomaly indicator.
