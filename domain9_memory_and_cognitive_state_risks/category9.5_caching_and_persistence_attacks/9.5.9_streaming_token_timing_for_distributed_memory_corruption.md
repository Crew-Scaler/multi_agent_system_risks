# 9.5.9 Streaming Token Timing for Distributed Memory Corruption [NoFW]

Attackers controlling streaming timing can inject context at specific moments when distributed multi-agent memory synchronizes. By controlling one agent's streaming speed, attackers force specific memory states in other agents through synchronization timing manipulation. Distributed multi-agent memory creates multiple synchronization windows absent from singular centralized memory, enabling targeted memory state control through timing attacks.
