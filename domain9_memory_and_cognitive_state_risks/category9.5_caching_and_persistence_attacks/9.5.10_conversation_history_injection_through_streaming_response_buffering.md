# 9.5.10 Conversation History Injection Through Streaming Response Buffering [NoFW]

In multi-agent streaming pipelines where Agent A streams output to Agent B, Agent B buffers to conversation history before stream completion. Attackers inject instructions mid-stream that get buffered to history before validation, which Agent B then reads as authoritative context. Unlike batch responses validated completely before buffering, streaming creates temporal injection windows at each agent handoff boundary.
