# 9.3.7 Audio Transcript Memory Corruption Through Whisper Hallucination Persistence [NoFW]

Transcribed audio segments containing ASR model hallucinations integrate into agent memory and persist across sessions. When agents retrieve transcripts in future interactions, hallucinated content appearing as legitimate transcript text triggers backdoor behaviors. Agents cannot distinguish hallucinated instructions from accurate transcription without validating against source audio, and multi-agent systems storing transcripts in shared episodic memory allow one hallucination to propagate as authoritative context across many agents.
