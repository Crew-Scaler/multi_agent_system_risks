# 9.3.10 Multi-Modal Web Benchmark Content Injection via Image Metadata [NoFW]

Web benchmarks including visual components create attack surfaces where adversaries embed instructions in image metadata (EXIF tags, XMP data, steganographic content) that vision models extract as text context. In multi-agent visual processing pipelines where one agent captures images and another processes their metadata separately, injected metadata is treated as trusted content because the processing agent has no visibility into the original image source.
