# 9.3.5 Distributed Memory Poisoning via Perception Corruption [NoFW]

Attackers corrupt perception modules injecting hallucinated facts that propagate through validation gates into long-term memory. In distributed multi-agent systems, sharing of poisoned memories causes corrupted facts to replicate across agent teams with elevated trust because they carry the implicit endorsement of peer agents. Distributed memory systems transform single perception errors into persistent, self-reinforcing collective false beliefs that are difficult to trace back to the original perception failure.
