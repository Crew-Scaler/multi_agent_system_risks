# 9.3.9 CSS Content Injection Through Pseudo-Element Rendering [NoFW]

CSS pseudo-elements (::before, ::after) allow content injection that creates dual-channel content: visual rendering shows benign content while content extraction reveals adversarial instructions. In multi-agent web systems with separate rendering and content extraction agents, pseudo-element instructions execute in the extraction agent's analysis pipeline without being visible in rendering context. This dual-channel property defeats review processes that rely on visual inspection to verify content safety.
