# 9.2.6 Knowledge Graph Relationship Injection via Agent Coordination [NoFW]

Malicious agents insert false multi-hop relationships into shared knowledge graphs, exploiting cross-agent traversal patterns to create spurious connections influencing collective decision-making through fabricated causal chains. An attacker with graph write access inserts false triples that downstream agents follow as established facts during traversal. In multi-agent systems sharing a single knowledge graph, one injected false relationship propagates across all agents' reasoning without requiring individual agent compromise, and detection requires comparing the live graph against a known-good baseline that multi-agent systems rarely perform per-query.
