# 9.2.3 Inter-Agent Memory Type Misclassification During State Handoff [NoFW]

Attackers exploit handoff points causing memory type misclassification, storing episodic events in Agent A that get retrieved as semantic knowledge by Agent B. Malicious prompts store time-bounded authorization events as episodic memory, but orchestrator passage reconstructs them as semantic facts without temporal qualifiers. Type confusion is uniquely multi-agent because serialized communication loses type fidelity through context summarization, and each handoff in a chain degrades type integrity further, transforming specific learned patterns into broad semantic truths applied universally.
