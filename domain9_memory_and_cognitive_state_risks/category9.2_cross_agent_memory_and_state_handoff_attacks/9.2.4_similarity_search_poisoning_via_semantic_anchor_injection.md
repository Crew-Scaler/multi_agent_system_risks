# 9.2.4 Similarity-Search Poisoning via Semantic Anchor Injection [NoFW]

Attackers inject carefully-crafted semantic anchor memories exploiting similarity-based retrieval to hijack future queries across networks through multiple modalities. Both text-based and multimodal semantic anchors place adversarial embeddings semantically proximate to legitimate operational queries, causing malicious context to be co-retrieved with legitimate queries. Multi-agent amplification occurs when poisoned semantic memory aggregates into shared knowledge bases affecting all downstream agents querying semantically similar content, creating a single point of failure in centralized vector stores.
