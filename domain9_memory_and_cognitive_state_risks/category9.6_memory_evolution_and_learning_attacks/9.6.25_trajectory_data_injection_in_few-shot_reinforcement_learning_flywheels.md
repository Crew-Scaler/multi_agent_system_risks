# 9.6.25 Trajectory Data Injection in Few-Shot Reinforcement Learning Flywheels [NoFW]

Agent trajectory learning creates feedback loops where successful behaviors become future demonstration examples across multi-agent chains. Adversaries inject initial malicious trajectories that complete assigned tasks superficially while embedding covert control-flow changes. The success signal reinforces the full trajectory including covert steps, causing downstream agents to inherit and propagate corrupted behavior patterns through the learning flywheel, amplifying the initial injection across all agents in the chain.
