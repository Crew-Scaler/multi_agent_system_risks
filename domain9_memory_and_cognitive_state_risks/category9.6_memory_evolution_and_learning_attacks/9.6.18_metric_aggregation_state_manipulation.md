# 9.6.18 Metric Aggregation State Manipulation [NoFW]

Evaluation pipelines aggregating metrics across test cases compute mean scores and failure rates from intermediate aggregation state. If aggregation state is mutable or intermediate results are stored, attackers with write access can manipulate the aggregation itself rather than individual results. Multi-agent metric aggregation where different agents contribute results to a central aggregation layer provides a high-leverage attack surface where one aggregation manipulation affects final metrics without altering any individual agent's reported results.
