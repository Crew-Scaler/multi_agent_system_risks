# 9.6.10 Baseline Metric Degradation as Silent Attack Activation [NoFW]

Efficiency baselines stored in memory become attack vectors when attackers gradually poison baseline references, creating pressure for agents to execute efficiency improvements. An agent targeting improvement over an artificially inflated baseline will execute any available efficiency measure—including attacker-controlled fallback routines with reduced validation. Multi-agent systems sharing efficiency baselines enable one baseline poisoning to affect all agents' optimization targets simultaneously, creating fleet-wide silent activation of attacker-pre-positioned code paths.
