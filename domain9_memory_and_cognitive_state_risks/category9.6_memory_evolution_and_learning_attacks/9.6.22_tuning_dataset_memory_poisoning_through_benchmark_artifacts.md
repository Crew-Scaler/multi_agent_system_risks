# 9.6.22 Tuning Dataset Memory Poisoning Through Benchmark Artifacts [NoFW]

Parameter tuning relying on shared representative test sets creates fleet-wide poisoning when those test sets contain adversarial examples or benchmark artifacts. Agents tuning on poisoned shared benchmark data memorize benchmark-specific patterns that later activate malicious behavior when real inputs match learned benchmark patterns. Multi-agent tuning on shared datasets enables single-dataset poisoning to corrupt all agents' learned memory simultaneously.
