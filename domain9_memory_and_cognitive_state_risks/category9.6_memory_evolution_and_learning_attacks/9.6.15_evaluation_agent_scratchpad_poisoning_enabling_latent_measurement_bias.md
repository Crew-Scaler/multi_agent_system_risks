# 9.6.15 Evaluation Agent Scratchpad Poisoning Enabling Latent Measurement Bias [NoFW]

Evaluation agents maintaining scratchpads tracking evaluated test cases and intermediate calculations are vulnerable when scratchpads are shared for coordination. Attackers poison scratchpad entries establishing false evaluation history that downstream agents trust as prior evaluation results, propagating malicious measurements without triggering duplicate testing detection. Shared evaluation scratchpads create persistent false evaluation records affecting all agents reading the shared scratchpad data.
