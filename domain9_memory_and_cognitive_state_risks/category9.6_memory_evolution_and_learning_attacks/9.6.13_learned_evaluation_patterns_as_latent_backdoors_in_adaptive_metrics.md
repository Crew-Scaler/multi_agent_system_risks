# 9.6.13 Learned Evaluation Patterns as Latent Backdoors in Adaptive Metrics [NoFW]

Adaptive evaluation metrics learning patterns from prior evaluations embed those learned patterns in agent memory, creating backdoor injection vectors. Attackers inject malicious patterns early causing evaluation agents to adjust expectations incorrectly. Multi-agent evaluation systems storing learned patterns in shared memory create persistence where backdoored expectations propagate to all agents accessing the pattern store, turning learned backdoors into systematic evaluation bias affecting all downstream uses.
