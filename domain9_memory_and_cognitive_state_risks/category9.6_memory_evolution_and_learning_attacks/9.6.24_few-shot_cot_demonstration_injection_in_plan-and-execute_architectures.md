# 9.6.24 Few-Shot CoT Demonstration Injection in Plan-and-Execute Architectures [NoFW]

Chain-of-thought demonstrations central to Plan-and-Execute multi-agent systems create poisoning vectors when adversarial examples teach incorrect decomposition patterns to the planning agent. Poisoned demonstrations show the planner decomposing legitimate tasks into steps that include security bypass sub-steps. Execution agents follow plans containing malicious steps as standard prerequisites without independently verifying the plan's security properties—a risk unique to multi-agent plan-and-execute architectures where planning and execution are separated.
