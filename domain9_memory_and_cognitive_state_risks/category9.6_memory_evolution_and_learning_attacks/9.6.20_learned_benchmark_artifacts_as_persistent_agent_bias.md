# 9.6.20 Learned Benchmark Artifacts as Persistent Agent Bias [NoFW]

Agents overfitting to benchmark artifacts memorize benchmark-specific patterns rather than generalizable capabilities. In multi-agent systems, agents sharing learned overfitting patterns through memory or training data propagate benchmark bias across the network. When Agent A learns benchmark artifacts and shares those patterns with Agent B, Agent B inherits overfitting as a learned capability, creating distributed cognitive corruption where benchmark-specific biases persist across agent networks and fail silently on real-world inputs deviating from benchmark distributions.
