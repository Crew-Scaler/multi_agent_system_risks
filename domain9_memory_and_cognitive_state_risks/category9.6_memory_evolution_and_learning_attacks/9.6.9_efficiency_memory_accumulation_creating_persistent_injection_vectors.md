# 9.6.9 Efficiency Memory Accumulation Creating Persistent Injection Vectors [NoFW]

Efficiency tracking stores historical metrics in memory for quick access across agent interactions. Attackers injecting false efficiency data into shared memory affect planning decisions across subsequent interactions and sessions. Single-agent efficiency memory poisoning affects one session, but multi-agent shared efficiency memory enables persistent backdoors where one injection affects planning decisions for hours or days across the entire agent fleet.
