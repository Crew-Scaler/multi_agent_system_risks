# 9.6.14 Cached Evaluation Results Poisoning Through Result Deduplication and Replay Attacks [NoFW]

Evaluation pipelines caching test case results for efficiency create poisoning targets in multi-agent systems with shared evaluator cache. Two attack vectors exist: (1) result deduplication poisoning where cache entries for identical test cases serve corrupted results to all querying agents simultaneously; (2) replay attacks where cached results from prior evaluation runs fabricate current performance. Evaluation systems rarely validate that retrieved cached results correspond to the queried test case, enabling both simultaneous multi-agent compromise through cache layer attacks.
