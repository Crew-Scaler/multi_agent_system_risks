# 9.6.26 Fine-Tuning Data Contamination Through Few-Shot Example Selection [NoFW]

Automated few-shot selection creating fine-tuning datasets becomes a multi-stage attack vector when the selection agent is poisoned. Attackers poison the example selection agent's heuristics to select subtly malicious but high-scoring examples, which then fine-tune the next agent in the pipeline. Multi-agent fine-tuning chains (Agent A selects, Agent B fine-tunes, Agent C consumes) enable control-flow attacks where demonstration poisoning compounds through subsequent fine-tuning stages, amplifying the initial selection attack.
