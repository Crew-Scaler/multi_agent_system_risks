# 9.6.16 Evaluation Dataset Memory Corruption Through Progressive Modification [NoFW]

Evaluation datasets stored in mutable form can be progressively corrupted through small modifications across multiple evaluation runs in multi-agent pipelines. Attackers insert small corruptions through normal modification workflows via preprocessing, augmentation, and cleaning agents. Multi-agent dataset modification pipelines create distributed corruption vectors where normal agent operations provide cover for progressive poisoning, unlike singular evaluation with transparent dataset immutability.
