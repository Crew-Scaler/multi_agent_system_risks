# 9.6.11 Cost Attribution Memory as Decision Context Poisoning [NoFW]

Efficiency systems tracking cost per agent, per task, and per workflow store attribution metadata that becomes decision context for subsequent planning. Poisoned cost records cause cost-based agent selection decisions to route work to unintended or attacker-controlled agents. Multi-agent cost attribution shared across agents enables one cost data poisoning to affect all downstream agent selection decisions in cost-aware orchestration systems.
