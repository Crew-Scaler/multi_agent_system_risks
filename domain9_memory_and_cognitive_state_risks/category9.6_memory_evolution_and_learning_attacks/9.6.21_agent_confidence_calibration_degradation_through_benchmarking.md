# 9.6.21 Agent Confidence Calibration Degradation Through Benchmarking [NoFW]

In multi-agent systems, calibration errors accumulate across agent iterations through confidence score propagation. An overconfident agent propagates miscalibration through downstream agents relying on its confidence scores, creating compound miscalibration where confidence becomes increasingly detached from actual accuracy through agent-to-agent propagation. Singular agent miscalibration remains isolated; multi-agent confidence propagation enables miscalibration cascading that survives indefinitely in shared system memory.
