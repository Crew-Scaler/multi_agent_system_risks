# 9.6.7 Poisoned Parameter History in Shared Conversation Memory [NoFW]

Multi-turn parameter extraction errors in early turns persist in shared conversation memory and bias later turns. In multi-agent systems sharing memory, downstream agents inherit poisoned parameter values extracted by upstream agents as ground truth, with no mechanism to signal that a prior agent's extraction was erroneous. Unlike single agents that can recover through re-extraction, multi-agent memory sharing amplifies action-level problems across multiple agents through shared state propagation.
