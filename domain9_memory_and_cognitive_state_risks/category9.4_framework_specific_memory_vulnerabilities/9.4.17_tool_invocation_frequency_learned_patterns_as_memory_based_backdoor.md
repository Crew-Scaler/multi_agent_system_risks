# 9.4.17 Tool Invocation Frequency Learned Patterns as Memory-Based Backdoor [NoFW]

Agents with learning systems storing tool usage frequency patterns create backdoors when memory is poisoned with malicious sequences disguised as learned best practices. Unlike direct instruction injection, pattern-based backdoors are invisible as seemingly-learned operational norms. Multi-agent memory sharing propagates learned pattern backdoors across agent boundaries, creating fleet-wide behavioral corruption from a single pattern injection.
