# 9.4.16 Tool Availability State Corruption Through Memory Backdoors [NoFW]

Agent memory including information about which tools are available creates latent backdoors when attackers inject false availability information causing subsequent agents to use wrong or malicious tools. Presenting a malicious tool as a modern replacement for a legitimate deprecated tool causes all agents in a multi-agent system sharing tool availability memory to route work to the attacker-controlled tool.
