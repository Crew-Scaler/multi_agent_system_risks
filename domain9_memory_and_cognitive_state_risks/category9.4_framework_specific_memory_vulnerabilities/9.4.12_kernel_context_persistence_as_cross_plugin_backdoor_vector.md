# 9.4.12 Kernel Context Persistence as Cross-Plugin Backdoor Vector [Semantic Kernel]

Semantic Kernel maintains execution context persisting across plugin invocations. Attackers poison context by manipulating plugin outputs, causing subsequent plugin invocations to inherit corrupted context and activate latent backdoors. Kernel context shared across plugins means compromising one plugin poisons context for all subsequent plugins in the workflow, creating a broad blast radius compared to isolated single-agent plugin execution.
