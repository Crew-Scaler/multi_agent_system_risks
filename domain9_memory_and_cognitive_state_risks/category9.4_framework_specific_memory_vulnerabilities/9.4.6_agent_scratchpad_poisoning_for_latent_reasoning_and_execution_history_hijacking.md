# 9.4.6 Agent Scratchpad Poisoning for Latent Reasoning and Execution History Hijacking [LangChain]

LangChain agent scratchpads (Thought-Action-Observation history) are vulnerable to poisoning where malicious entries bias subsequent reasoning cycles and tool selections. Two attack vectors exist: (1) reasoning hijacking through injected entries appearing as established facts in reasoning history; (2) execution history fabrication through false prior outcomes causing downstream tool selections based on poisoned success records. Multi-agent systems sharing scratchpad state through session coordination enable cross-agent poisoning where one compromised scratchpad affects all downstream agents.
