# 9.4.10 CrewAI Task Context Persistence Enabling Cross-Task Latent Instruction Propagation [CrewAI]

CrewAI's hierarchical system with persistent task context creates opportunities for latent instructions injected early in task sequences to activate later through context inheritance. Managers receiving poisoned input propagate context forward through delegation chains, and the task abstraction creates an assumption that context inheritance represents legitimate prior analysis. Single early injection propagates deterministically through all dependent tasks without re-validation, a risk absent from independent single-agent task execution.
