# 9.4.5 Framework-Orchestrated Session Resumption Creating Distributed Memory Corruption [LangChain, LangGraph, AutoGen]

Frameworks implementing session persistence (LangChain, LangGraph, AutoGen) create long-lived cognitive state vulnerable to cross-framework poisoning during simultaneous resumption. When multiple frameworks restore state concurrently in multi-agent systems, poisoned state from one framework contaminates others through shared context—corrupted LangChain conversation history passed to LangGraph nodes becomes corrupted routing state. Unlike singular systems with one state persistence model, multi-agent combinations with multiple heterogeneous restoration mechanisms create inconsistencies enabling latent backdoors that persist across framework boundaries.
