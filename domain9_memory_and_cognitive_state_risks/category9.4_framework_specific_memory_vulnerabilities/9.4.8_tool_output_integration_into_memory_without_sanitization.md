# 9.4.8 Tool Output Integration into Memory Without Sanitization [LangChain]

Tool outputs in LangChain agents are incorporated directly into agent scratchpads and conversation history without sanitization before integration into persistent memory. This enables attackers to return tool results containing instructions that persist in agent memory and activate as backdoors in future tasks. Multi-agent systems where tool outputs from one agent become persistent memory affecting multiple downstream agents enable tool output poisoning with multiplicative reach across the entire agent network.
