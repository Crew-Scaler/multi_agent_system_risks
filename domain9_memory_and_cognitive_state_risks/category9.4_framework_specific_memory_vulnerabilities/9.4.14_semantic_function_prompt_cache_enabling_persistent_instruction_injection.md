# 9.4.14 Semantic Function Prompt Cache Enabling Persistent Instruction Injection [Semantic Kernel]

Semantic functions execute LLM inference on cached templates. When LLM responses are cached without sanitization, malicious responses cache persistently and influence all future semantic function executions with identical context. Shared semantic function caches across agents mean one injection poisons responses for all agents calling that function, creating fleet-wide persistent instruction injection through the shared cache layer.
