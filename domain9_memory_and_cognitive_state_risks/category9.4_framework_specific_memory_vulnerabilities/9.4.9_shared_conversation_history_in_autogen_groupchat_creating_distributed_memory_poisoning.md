# 9.4.9 Shared Conversation History in AutoGen GroupChat Creating Distributed Memory Poisoning [AutoGen]

AutoGen GroupChat maintains shared message history visible to all participants, but this shared history creates vulnerabilities where poisoning one agent's contributions affects all downstream agents equally. Early injections into shared history propagate to all agents throughout the conversation lifecycle. Unlike singular agent memory confined to one context, GroupChat shared history enables attackers to achieve synchronized compromise of all agents through a single-point poisoning rather than requiring individual agent compromise.
