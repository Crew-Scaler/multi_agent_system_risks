# 9.4.11 Manager Decision Memory in CrewAI Hierarchies Creating Cross-Worker Context Poisoning [CrewAI]

CrewAI managers maintain memory of worker outputs and decisions, but this memory becomes vulnerable when poisoned worker outputs remain in manager context and influence future delegations. Managers learning from corrupted worker outputs incorporate malicious patterns into future task delegation decisions. Attackers exploit this by causing systematic misalignment in delegation patterns through manager memory poisoning, a risk unique to hierarchical multi-agent frameworks with persistent manager-level state.
