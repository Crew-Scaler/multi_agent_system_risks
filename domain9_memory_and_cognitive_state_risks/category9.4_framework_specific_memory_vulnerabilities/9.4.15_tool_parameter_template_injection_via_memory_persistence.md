# 9.4.15 Tool Parameter Template Injection via Memory Persistence [NoFW]

Tool invocations using templated parameters (SQL queries, API endpoints, command arguments) stored in memory for efficiency create a vector where memory poisoning with malicious parameter templates causes subsequent tool invocations to execute with template injection. SQL injection templates persisting in shared multi-agent memory cause database queries to drift toward attacker objectives across all agents querying that memory, amplifying single-agent poisoning to fleet-wide impact.
