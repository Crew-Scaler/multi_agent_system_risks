# 12.3.16 Working Memory Phase Transitions Creating Monitoring Blind Spots [NoFW]

Working memory lifecycle phase transitions (assembly, processing, reasoning, generation, disposal) create discrete observability gaps; the disposal phase erases intermediate reasoning states and tool invocation evidence before monitoring can capture them, and attackers who time injection to the generation phase exploit the window between reasoning completion and disposal.
