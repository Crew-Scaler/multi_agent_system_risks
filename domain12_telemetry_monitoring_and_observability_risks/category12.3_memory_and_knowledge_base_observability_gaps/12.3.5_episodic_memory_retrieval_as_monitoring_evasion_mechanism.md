# 12.3.5 Episodic Memory Retrieval as Monitoring Evasion Mechanism [NoFW]

Retrieved episodic memories influence agent behavior without appearing in instruction logs; monitoring systems observe the resulting action but cannot trace it to the specific retrieved episode that motivated it, making memory-injection attacks appear as unprompted agent decisions.
