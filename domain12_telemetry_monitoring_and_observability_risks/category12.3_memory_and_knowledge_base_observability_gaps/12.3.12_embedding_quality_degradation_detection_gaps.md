# 12.3.12 Embedding Quality Degradation Detection Gaps [NoFW]

Cumulative embedding corruption from poisoned ingestion degrades retrieval accuracy gradually, but no monitoring metric directly measures embedding quality without running retrieval benchmark evaluations; quality degradation remains invisible until it produces noticeable downstream accuracy failures.
