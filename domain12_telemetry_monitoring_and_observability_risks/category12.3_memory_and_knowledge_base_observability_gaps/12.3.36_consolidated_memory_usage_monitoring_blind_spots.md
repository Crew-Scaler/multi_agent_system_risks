# 12.3.36 Consolidated Memory Usage Monitoring Blind Spots [NoFW]

Consolidated memory access patterns—which summaries were retrieved, when, and by which agent—are not tracked at the consolidation-entry level; attackers exploiting backdoored consolidated memories generate no anomalous telemetry if monitoring only observes final agent outputs rather than which consolidated entries influenced them.
