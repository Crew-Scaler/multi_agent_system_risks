# 12.3.24 Pipeline Transformation Throughput Monitoring Blind Spots for Chunk-Level Operations [NoFW]

ETL monitoring tracks document-level throughput (documents per second) without measuring chunk-level throughput; semantic chunking complexity varies dramatically per document, and document-level metrics can appear healthy while chunk-level processing throughput has degraded by an order of magnitude due to chunking algorithm performance regression.
