# 12.3.27 Cache Performance Monitoring Gaps Obscuring Per-Layer Hit Rate Degradation and Eviction Pressure [NoFW]

RAG systems implement multiple independent caching layers (query embedding cache, semantic cache, context cache, response cache) each with distinct hit rate characteristics; aggregate cache hit rate metrics mask per-layer performance divergence where one layer is thrashing while others appear healthy, and eviction pressure on specific layers is invisible until cache churn causes observable latency degradation.
