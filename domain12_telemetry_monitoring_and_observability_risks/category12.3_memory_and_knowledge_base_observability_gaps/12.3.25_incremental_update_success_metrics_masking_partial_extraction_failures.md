# 12.3.25 Incremental Update Success Metrics Masking Partial Extraction Failures [NoFW]

ETL incremental update runs report success when at least some sources extract successfully, hiding complete extraction failures for individual sources; when one source fails for multiple consecutive runs, monitoring shows apparently successful runs with declining document counts that operators attribute to natural source update frequency variation rather than systematic extraction failures.

## Supporting Evidence
