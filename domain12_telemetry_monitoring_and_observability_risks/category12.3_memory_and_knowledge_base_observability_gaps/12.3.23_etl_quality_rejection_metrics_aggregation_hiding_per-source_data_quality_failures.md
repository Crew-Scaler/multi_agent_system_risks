# 12.3.23 ETL Quality Rejection Metrics Aggregation Hiding Per-Source Data Quality Failures [NoFW]

ETL quality validation monitoring reports aggregate rejection rates across all data sources without per-source breakdowns; a single source experiencing severe quality degradation (such as a Confluence space generating auto-generated low-quality content) can be hidden behind high-quality metrics from other sources, delaying detection of source-specific data quality failures.
