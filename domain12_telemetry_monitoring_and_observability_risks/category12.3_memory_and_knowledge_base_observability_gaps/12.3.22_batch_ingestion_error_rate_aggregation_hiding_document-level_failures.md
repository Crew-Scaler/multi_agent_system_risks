# 12.3.22 Batch Ingestion Error Rate Aggregation Hiding Document-Level Failures [NoFW]

Batch ingestion monitoring tracks batch-level success rates; a batch where 95 of 100 documents succeed is counted as successful, hiding the 5% document failure rate; across multiple batches, this aggregation can conceal double-digit document-level failure rates behind near-perfect batch success metrics.
