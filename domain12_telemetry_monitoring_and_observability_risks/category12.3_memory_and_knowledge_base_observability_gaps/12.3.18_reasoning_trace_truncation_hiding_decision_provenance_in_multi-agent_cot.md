# 12.3.18 Reasoning Trace Truncation Hiding Decision Provenance in Multi-Agent CoT [NoFW]

Output token constraints cause chain-of-thought reasoning traces to be truncated before completion in multi-agent systems; downstream agents inherit decisions whose justifications were discarded, and monitoring systems analyzing token efficiency metrics interpret aggressive truncation as a positive efficiency signal while losing the decision provenance needed to audit safety.
