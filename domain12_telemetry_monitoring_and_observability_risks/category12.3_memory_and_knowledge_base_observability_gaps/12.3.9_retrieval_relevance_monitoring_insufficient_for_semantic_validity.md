# 12.3.9 Retrieval Relevance Monitoring Insufficient for Semantic Validity [NoFW]

Monitoring tracks embedding similarity scores as a proxy for retrieval quality but does not validate semantic appropriateness of retrieved content for the query context; a highly relevant document by similarity score may contain contextually inappropriate instructions, and monitoring based on relevance metrics provides false assurance about retrieval safety.
