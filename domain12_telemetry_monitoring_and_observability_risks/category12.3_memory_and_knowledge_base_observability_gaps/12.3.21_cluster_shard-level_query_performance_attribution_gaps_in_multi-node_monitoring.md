# 12.3.21 Cluster Shard-Level Query Performance Attribution Gaps in Multi-Node Monitoring [NoFW]

Multi-node vector database clusters expose aggregate cluster-level query latency but not shard-level or node-level performance breakdowns; performance degradation on specific shards due to poisoned index rebuilds or resource contention is masked in aggregate cluster metrics, and per-agent performance asymmetries caused by shard distribution are invisible to monitoring that cannot attribute query performance to specific shards.
