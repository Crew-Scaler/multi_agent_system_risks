# 12.3.29 Quality Dashboard Aggregation Hiding Per-Agent Validation Failure Distribution Patterns [NoFW]

Fleet-wide quality dashboards display aggregate pass rates across all agents without per-agent breakdowns; an agent achieving 94% pass rate and an agent achieving 48% pass rate produce an aggregate of 71% that appears acceptable, hiding that one agent has severe quality issues requiring investigation while the other performs well.
