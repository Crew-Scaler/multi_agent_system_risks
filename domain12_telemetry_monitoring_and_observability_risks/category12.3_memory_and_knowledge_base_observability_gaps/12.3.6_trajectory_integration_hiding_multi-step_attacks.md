# 12.3.6 Trajectory Integration Hiding Multi-Step Attacks [NoFW]

Trajectories that aggregate multi-step solution paths present monitoring with only final outcomes, concealing malicious intermediate steps embedded early in the trajectory that appeared to be legitimate problem-solving actions at time of execution.
