# 12.3.7 Metadata Filtering Complexity as Detection Evasion [NoFW]

Complex retrieval metadata filters combine temporal, contextual, and frequency-based conditions in ways that monitoring systems cannot exhaustively track; adversaries craft poisoned memory entries with retrieval conditions specifically designed to activate only under metadata combinations that fall outside instrumented monitoring coverage.
