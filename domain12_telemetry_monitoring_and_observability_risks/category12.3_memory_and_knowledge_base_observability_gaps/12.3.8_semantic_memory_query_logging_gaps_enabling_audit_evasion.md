# 12.3.8 Semantic Memory Query Logging Gaps Enabling Audit Evasion [NoFW]

RAG query logging is selectively disabled or sampled for performance reasons; adversaries who probe the logging coverage can identify which query patterns are not recorded and craft attack queries that consistently avoid instrumented paths, leaving no audit trail.

## Supporting Evidence

- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
