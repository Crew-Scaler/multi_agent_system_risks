# 12.3.11 Temporal Validity Enforcement Blind Spots [NoFW]

Knowledge base entries carry temporal validity metadata, but monitoring does not verify that retrieving agents check this metadata before using retrieved content; stale documents past their validity date remain retrievable and are used without the monitoring system detecting the temporal validity violation.
