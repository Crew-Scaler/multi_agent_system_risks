# 12.3.35 Knowledge Graph Evolution Telemetry Opacity [NoFW]

Knowledge graph updates lack detailed telemetry tracking why relationships were modified or who authorized specific changes, leaving minimal audit trails for attackers injecting relationships; in multi-agent systems with shared knowledge graphs, the collective update pattern creates plausible deniability when multiple agents query and update simultaneously, enabling attackers to inject relationships that appear to result from legitimate multi-agent consensus rather than adversarial manipulation.
