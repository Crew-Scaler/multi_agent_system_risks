# 12.3.14 Cache Hit Rate Manipulation as Performance Monitoring Bypass [NoFW]

Attackers craft queries designed to hit cached entries containing malicious content, inflating cache hit rates and maintaining healthy performance metrics while consistently retrieving adversarially chosen responses; monitoring that interprets high hit rates as a positive signal is deceived into false assurance.
