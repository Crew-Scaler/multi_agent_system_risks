# 12.3.30 Batch Processing Monitoring Gaps Obscuring Within-Batch Document-Level Failure Patterns [NoFW]

Batch-level monitoring records whether a batch succeeded or failed overall; within-batch document-level failures (quality validation rejections, duplicate detection failures, individual document processing errors) are invisible to batch-level metrics, and partial retry logic that reprocesses entire batches creates duplicate ingestion that monitoring attributes to retry success rather than correctness failure.
