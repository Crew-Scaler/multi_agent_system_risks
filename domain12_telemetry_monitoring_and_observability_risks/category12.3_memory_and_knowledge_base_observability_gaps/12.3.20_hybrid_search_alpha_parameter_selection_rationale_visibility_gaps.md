# 12.3.20 Hybrid Search Alpha Parameter Selection Rationale Visibility Gaps [NoFW]

Hybrid search alpha parameters controlling vector-versus-keyword weighting are logged as final selected values without capturing the selection rationale; adversaries who manipulate the alpha selection logic (such as an ML classifier) to systematically choose values that eliminate beneficial retrieval modalities remain undetected because monitoring shows only the selected value, not whether that selection was appropriate for the query type.
