# 12.3.19 Hierarchical Compression Observation Opaqueness Creating Semantic Monitoring Failures [NoFW]

Hierarchical memory compression stores working summaries in active memory while full conversation history is stored in external databases rarely inspected for anomalies; security monitoring of working memory summaries misses semantic content present only in full histories, and coordinated cross-agent attacks that are visible only when comparing full histories from multiple agents remain undetected in summary-level monitoring.
