# 12.4.9 GPU-Level DCGM Telemetry Aggregation Exposing Multi-Agent Inference Capacity and Resource Utilization Patterns [NoFW]

DCGM telemetry aggregated from multi-agent inference infrastructure reveals per-GPU utilization, memory allocation, and thermal headroom; adversaries accessing these dashboards learn exact saturation thresholds and peak-hour headroom, enabling precisely calibrated denial-of-inference attacks timed to periods of minimal remaining capacity.
