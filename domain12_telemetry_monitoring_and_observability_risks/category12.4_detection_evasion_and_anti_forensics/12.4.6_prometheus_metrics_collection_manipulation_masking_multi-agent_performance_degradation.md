# 12.4.6 Prometheus Metrics Collection Manipulation Masking Multi-Agent Performance Degradation [NoFW]

Adversaries with Kubernetes API access can intercept Prometheus scraper traffic or modify ServiceMonitor selectors to exclude degraded pods from monitoring coverage; histogram bucket value manipulation causes percentile calculations to report artificially healthy latencies while actual user-facing performance has severely degraded, and in multi-agent fleets this manipulation can be applied fleet-wide through centralized scraper traffic interception.
