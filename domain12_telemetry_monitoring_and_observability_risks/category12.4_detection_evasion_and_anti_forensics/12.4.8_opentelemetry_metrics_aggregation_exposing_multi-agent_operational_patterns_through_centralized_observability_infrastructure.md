# 12.4.8 OpenTelemetry Metrics Aggregation Exposing Multi-Agent Operational Patterns Through Centralized Observability Infrastructure [NoFW]

Centralized OpenTelemetry dashboards aggregating metrics from all agents expose fleet-wide dependency patterns, tool invocation frequencies, cost structures, and performance headroom to any adversary who accesses the observability system; this intelligence enables precise attack calibration (targeting during peak hours, sizing denial-of-wallet attacks to known cost structures) that would be unavailable from monitoring a single agent.
