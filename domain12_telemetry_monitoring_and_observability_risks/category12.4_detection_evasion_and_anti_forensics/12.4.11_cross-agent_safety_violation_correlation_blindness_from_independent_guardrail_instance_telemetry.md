# 12.4.11 Cross-Agent Safety Violation Correlation Blindness from Independent Guardrail Instance Telemetry [NoFW]

Per-agent guardrail systems log safety violations to independent telemetry streams; a distributed jailbreak campaign submitting 2–3 attempts per agent across a 45-agent fleet generates per-agent violation rates of approximately 1% (below individual agent anomaly thresholds) while constituting a 100-attempt coordinated campaign that would be detectable only through cross-fleet violation correlation.
