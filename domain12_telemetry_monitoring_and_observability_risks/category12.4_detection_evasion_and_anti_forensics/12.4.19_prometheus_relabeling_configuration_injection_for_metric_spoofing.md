# 12.4.19 Prometheus Relabeling Configuration Injection for Metric Spoofing [NoFW]

Prometheus relabeling rules transform metric labels, and if relabeling configuration comes from untrusted sources, attackers inject malicious relabel rules causing metric spoofing. A relabel rule "target_label: agent_identity, replacement: trusted_agent" causes all subsequent metrics to appear as originating from trusted_agent despite coming from a compromised agent.
