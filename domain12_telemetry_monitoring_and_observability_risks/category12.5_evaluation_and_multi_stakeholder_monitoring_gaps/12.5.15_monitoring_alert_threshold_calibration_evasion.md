# 12.5.15 Monitoring Alert Threshold Calibration Evasion [NoFW]

Continuous evaluation monitoring uses static alert thresholds that adversaries can evade by causing gradual metric degradation that stays within threshold bounds; the cumulative effect of many small sub-threshold degradations across multiple metrics and agents exceeds the practical quality bounds without triggering any discrete alerts.
