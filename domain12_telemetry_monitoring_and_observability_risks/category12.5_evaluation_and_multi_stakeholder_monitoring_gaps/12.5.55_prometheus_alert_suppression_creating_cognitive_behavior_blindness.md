# 12.5.55 Prometheus Alert Suppression Creating Cognitive Behavior Blindness [NoFW]

Prometheus alert deduplication suppresses repeated firings of the same alert after its initial occurrence; recurring issues that resolve and re-trigger appear as isolated single events to monitoring, and in multi-agent orchestration systems where orchestrators use alert presence or absence for decision-making, alert suppression causes orchestrators to assume resolved issues while underlying problems persist, propagating incorrect assumptions across all dependent agents.
