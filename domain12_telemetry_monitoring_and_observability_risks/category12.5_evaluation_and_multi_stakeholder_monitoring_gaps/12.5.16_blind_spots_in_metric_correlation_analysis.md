# 12.5.16 Blind Spots in Metric Correlation Analysis [NoFW]

Evaluation monitoring analyzes each metric independently without computing cross-metric correlations; coordinated attacks that simultaneously degrade multiple metrics in patterns that appear statistically uncorrelated when analyzed individually reveal their adversarial coordination only through multivariate correlation analysis that current monitoring does not perform.
