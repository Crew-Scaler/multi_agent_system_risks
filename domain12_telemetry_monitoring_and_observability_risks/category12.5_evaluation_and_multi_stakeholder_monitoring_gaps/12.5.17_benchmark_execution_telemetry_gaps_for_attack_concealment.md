# 12.5.17 Benchmark Execution Telemetry Gaps for Attack Concealment [NoFW]

Multi-agent benchmark evaluation distributes telemetry across per-agent log streams with natural gaps at agent communication boundaries; attacks that inject malicious instructions from one agent to another during benchmark execution leave traces in each agent's log independently but leave no trace in any log of the cross-agent instruction passage itself.
