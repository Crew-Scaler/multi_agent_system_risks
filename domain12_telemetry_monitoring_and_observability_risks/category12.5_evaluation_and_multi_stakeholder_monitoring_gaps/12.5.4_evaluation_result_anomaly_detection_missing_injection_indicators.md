# 12.5.4 Evaluation Result Anomaly Detection Missing Injection Indicators [NoFW]

Evaluation monitoring tracks metric trend anomalies using threshold-based detection but does not semantically analyze evaluation results for injection signatures; a sudden improvement in all metrics following a metric definition update is not flagged as a suspicious pattern even though it may indicate that metric calculation instructions were injected.

## Supporting Evidence

- Yupei Liu, Yuqi Jia, Runpeng Geng, Jinyuan Jia, and Neil Zhenqiang Gong. 2024. Formalizing and Benchmarking Prompt Injection Attacks and Defenses. In *USENIX Security Symposium 2024; the model sizes for closed-source models are from blog posts*. https://arxiv.org/abs/2310.12815
