# 12.5.102 Streaming Telemetry Collection Gaps Creating Real-Time Attack Opacity [NoFW]

Streaming responses create telemetry collection challenges where complete analysis is only possible after streaming finishes; real-time monitoring must choose between analyzing incomplete partial streams (risking false positives) or waiting for completion (losing detection responsiveness), and in multi-agent systems with multiple simultaneous streaming agents, the temporal lag between streaming completion and monitoring analysis creates windows where attacks inject content appearing benign in partial streams but malicious in complete form.
