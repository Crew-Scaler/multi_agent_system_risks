# 12.5.62 Observable Metrics Obscuring Non-Observable Internal Agent States [NoFW]

External monitoring observes latency, throughput, and error rates but cannot directly observe internal agent states such as reasoning validity, confidence calibration, or memory corruption; internal state corruption in one agent propagates through the multi-agent workflow invisibly, only becoming detectable through external metric degradation after the corruption has cascaded through multiple downstream agents.
