# 12.5.90 Caching Creating Hit/Miss Monitoring Blind Spots [NoFW]

Cache hits bypass real-time tool invocation and RAG pipeline execution, making those operations invisible to tool invocation monitoring; an agent that primarily queries cached results appears to have no tool usage in monitoring while actually relying on cached outputs from previous tool invocations, and poisoned cached content drives agent behavior without generating any current-period monitoring signal.

## Supporting Evidence
