# 12.5.54 Message Queue Throughput Metrics Hiding Quality Degradation [NoFW]

Message queue throughput metrics measuring messages-per-second appear healthy when high-throughput poisoned messages are processed; agents maintaining excellent throughput while processing malicious content report strong queue health indicators, and in multi-agent shared queues all consuming agents process poisoned content while aggregate throughput metrics remain healthy.
