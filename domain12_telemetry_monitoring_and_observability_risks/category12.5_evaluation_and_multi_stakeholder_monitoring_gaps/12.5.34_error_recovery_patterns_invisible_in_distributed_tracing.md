# 12.5.34 Error Recovery Patterns Invisible in Distributed Tracing [NoFW]

Error recovery in multi-agent systems distributes across agents (Agent A detects failure, Agent B implements retry, Agent C verifies recovery); per-agent distributed tracing captures each agent's local recovery actions but not the cross-agent recovery orchestration, creating "recovery blindness" where monitoring cannot determine whether distributed recovery actually restored the system to a correct state.
