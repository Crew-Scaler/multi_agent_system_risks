# 12.5.99 Tool Invocation Causality Tracking Failures in Asynchronous Execution [NoFW]

Tool frameworks enable asynchronous tool invocation where agents queue calls without waiting for completion; in multi-agent systems, causality attribution for which agent initiated which tool becomes unclear when Tool invocation by Agent B was actually initiated by Agent A in a prior step through asynchronous request queuing, and monitoring systems that attribute tool invocations by immediate requester rather than original initiator enable attackers to spoof causality attribution in audit logs.
