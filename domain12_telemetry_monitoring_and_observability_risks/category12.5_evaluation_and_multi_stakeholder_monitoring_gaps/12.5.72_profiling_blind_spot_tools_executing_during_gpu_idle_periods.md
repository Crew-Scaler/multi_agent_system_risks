# 12.5.72 Profiling Blind Spot: Tools Executing During GPU Idle Periods [NoFW]

GPU profiling captures compute utilization but does not capture tool execution behavior during GPU idle periods; adversaries time malicious tool invocations to occur during GPU idle intervals knowing that profiling coverage is focused on compute-intensive operations, and in multi-agent deployments where some agents have idle GPU periods at different times, rotating tool execution through idle periods across agents evades temporal profiling coverage.
