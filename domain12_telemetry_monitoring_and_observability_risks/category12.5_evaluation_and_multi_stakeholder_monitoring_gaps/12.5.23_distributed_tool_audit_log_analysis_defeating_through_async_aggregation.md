# 12.5.23 Distributed Tool Audit Log Analysis Defeating Through Async Aggregation [NoFW]

Tool audit logs in multi-agent systems are aggregated asynchronously with delays; adversaries time attacks to complete while logs have not yet been aggregated, or craft attack sequences that appear benign when analyzed in isolated time windows but reveal malicious intent only through temporal correlation across agents that asynchronous aggregation prevents.
