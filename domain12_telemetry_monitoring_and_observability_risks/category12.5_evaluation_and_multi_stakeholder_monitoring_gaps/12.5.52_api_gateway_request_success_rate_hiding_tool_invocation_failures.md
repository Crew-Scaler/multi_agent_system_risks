# 12.5.52 API Gateway Request Success Rate Hiding Tool Invocation Failures [NoFW]

API gateway success metrics track HTTP response codes but not whether returned responses indicate tool-level success; tools that return HTTP 200 with error payload bodies are counted as successes in gateway monitoring, and in multi-agent systems where gateway metrics are the primary observability layer, tool invocation failures that return valid HTTP responses are systemically hidden.
