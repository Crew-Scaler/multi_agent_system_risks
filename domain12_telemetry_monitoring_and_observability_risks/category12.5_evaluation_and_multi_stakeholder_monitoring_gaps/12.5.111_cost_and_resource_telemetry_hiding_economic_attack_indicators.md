# 12.5.111 Cost and Resource Telemetry Hiding Economic Attack Indicators [NoFW]

Token consumption and API cost telemetry is treated as an operational metric rather than a security signal, creating blind spots for denial-of-wallet attacks, resource exhaustion through agent cascade, and attacker-induced expensive computation.

## Supporting Evidence

- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../../papers/2603.11088.md)
  - "Attackers exploit this autonomy to trigger costly API calls, force infinite execution loops, or cause excessive memory use, creating denial-of-service conditions that can render the agent and external systems unusable or economically unsustainable."
