# 12.5.45 Efficiency Metric Correlation Analysis Blind Spot [NoFW]

Efficiency monitoring tracks metrics such as latency, token count, and API calls in isolation; coordinated attacks that simultaneously elevate multiple metrics (high latency plus high tokens plus unusual API call patterns) reveal attack signatures only through cross-metric correlation that single-metric threshold monitoring does not perform.
