# 12.5.31 Parameter Validation Failures Undetected in Aggregated Metrics [NoFW]

Aggregate action accuracy metrics hide per-parameter-type failure patterns; an agent with 60% accuracy on temporal parameters and 92% accuracy on other types appears as 87% overall accuracy, masking systematic temporal parameter weakness that attackers can exploit knowing that particular parameter class evades detection in aggregate metrics.
