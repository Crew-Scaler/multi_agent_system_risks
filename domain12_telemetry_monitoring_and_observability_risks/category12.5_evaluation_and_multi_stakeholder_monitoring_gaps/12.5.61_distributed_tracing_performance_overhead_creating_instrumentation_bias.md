# 12.5.61 Distributed Tracing Performance Overhead Creating Instrumentation Bias [NoFW]

Comprehensive distributed tracing adds latency overhead that varies across agents based on instrumentation depth; heavily-instrumented agents exhibit different latency profiles than lightly-instrumented agents, creating cross-agent behavior divergence in monitored environments that prevents accurate verification of multi-agent interaction effects in production.
