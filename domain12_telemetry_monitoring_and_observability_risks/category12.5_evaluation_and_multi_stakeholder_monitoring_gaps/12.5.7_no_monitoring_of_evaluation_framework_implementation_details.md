# 12.5.7 No Monitoring of Evaluation Framework Implementation Details [NoFW]

Different evaluation frameworks (custom Python scripts, LLM-based evaluators, automated testing systems) produce results without any monitoring visibility into framework behavior or implementation; LLM-based evaluators can be compromised through prompt injection into the evaluator's context without any logging evidence distinguishing legitimate evaluation from manipulated evaluation.
