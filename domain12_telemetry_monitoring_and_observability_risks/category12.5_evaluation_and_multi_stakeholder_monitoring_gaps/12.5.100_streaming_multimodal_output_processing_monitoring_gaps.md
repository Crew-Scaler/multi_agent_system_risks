# 12.5.100 Streaming Multimodal Output Processing Monitoring Gaps [NoFW]

Multimodal synthesis produces streaming outputs combining text, image references, and audio links in multi-agent systems where dependent agents process partial outputs before the complete stream is generated; this creates intermediate states where image references streaming before malicious text instructions enable dependent agents to act on incomplete content, and monitoring focused on final streaming outcomes misses attacks exploiting intermediate multimodal state ordering.
