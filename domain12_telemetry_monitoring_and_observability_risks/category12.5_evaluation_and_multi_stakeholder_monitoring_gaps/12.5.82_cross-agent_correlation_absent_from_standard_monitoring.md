# 12.5.82 Cross-Agent Correlation Absent from Standard Monitoring [NoFW]

Standard Prometheus and Kubernetes monitoring collect per-service metrics without cross-service correlation capabilities; coordinated attacks that manifest as Agent A latency spikes correlating with Agent B failures (indicating attack propagation) appear in monitoring as two independent service events rather than as a correlated attack pattern, preventing detection of coordinated multi-agent campaigns through standard monitoring stacks.

## Supporting Evidence

- Chen Yueh-Han, Nitish Joshi, Yulin Chen, Maksym Andriushchenko, Rico Angell, and He He. 2026. Monitoring Decomposition Attacks in LLMs with Lightweight Sequential Monitors. In *International Conference on Learning Representations (ICLR 2026)*. https://arxiv.org/abs/2506.10949
