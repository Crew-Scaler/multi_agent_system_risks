# 12.5.49 Message Queue Consumer Lag Hiding Agent Processing Failures [NoFW]

Message queue consumer lag metrics indicate pending message backlogs but do not distinguish between high-throughput processing and failed processing; agents experiencing processing failures accumulate lag indistinguishably from agents experiencing high load, and in shared queues serving multiple agents, per-agent failure rates are invisible in aggregate consumer lag metrics.
