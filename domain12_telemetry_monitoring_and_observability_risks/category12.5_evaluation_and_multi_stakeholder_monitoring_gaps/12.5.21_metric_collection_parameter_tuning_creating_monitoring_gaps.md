# 12.5.21 Metric Collection Parameter Tuning Creating Monitoring Gaps [NoFW]

Agents are instrumented with heterogeneous monitoring parameters (some with comprehensive trace logging, others with minimal logging for performance); adversaries preferentially route malicious operations through agents with reduced instrumentation, exploiting monitoring-parameter asymmetry to avoid detection.

## Supporting Evidence

- Yusheng Zheng, Yanpeng Hu, Tong Yu, and Andi Quinn. 2025. AgentSight: System-Level Observability for AI Agents Using eBPF. In *PACMI'2025*. https://doi.org/10.1145/3766882.3767169
  - [→ 2508.02736.md](../../papers/2508.02736.md)
  - "This creates a critical semantic gap: existing tools observe either an agent's high-level intent (via LLM prompts) or its low-level actions (e.g., system calls), but cannot correlate these two views."
