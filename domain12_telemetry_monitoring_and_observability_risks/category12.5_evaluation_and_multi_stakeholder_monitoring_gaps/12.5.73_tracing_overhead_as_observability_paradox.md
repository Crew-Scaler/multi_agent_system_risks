# 12.5.73 Tracing Overhead as Observability Paradox [NoFW]

Distributed tracing instrumentation adds latency overhead that causes production behavior (without tracing) to differ from monitored behavior (with tracing), creating observability paradox where the act of monitoring changes what is being observed; in multi-agent systems where some agents are instrumented and others are not, behavior comparisons across agents conflate application logic differences with instrumentation overhead differences.
