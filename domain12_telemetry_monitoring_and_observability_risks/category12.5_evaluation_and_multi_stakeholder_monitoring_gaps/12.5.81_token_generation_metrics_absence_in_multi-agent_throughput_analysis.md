# 12.5.81 Token Generation Metrics Absence in Multi-Agent Throughput Analysis [NoFW]

Inference monitoring reports requests-per-second without tracking tokens-per-second; adversaries craft prompts that produce minimal token outputs (appearing as high-throughput healthy operations) while consuming disproportionate compute through expensive reasoning operations, and monitoring that tracks only request throughput cannot detect this compute-to-output ratio manipulation.
