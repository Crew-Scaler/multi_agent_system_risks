# 12.5.66 Tool Invocation Audit Log Blind Spots in Service Mesh [NoFW]

Service mesh network logs capture connection endpoints and byte counts but not semantic content of agent-to-agent tool invocations; attackers routing malicious instructions through service mesh connections evade semantic audit coverage, as mesh logs show "Agent A called Agent B's tool endpoint" without logging the instruction content or parameters passed in the call.
