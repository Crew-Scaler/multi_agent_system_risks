# 12.5.29 Cross-Agent Context Correlation Blindness in Anomaly Detection [NoFW]

Per-agent anomaly detection identifies unusual tool invocation patterns within each agent's individual behavior stream; adversaries distribute anomalous invocations across multiple agents so that each agent's individual pattern appears normal, while the cross-agent correlation that would reveal the coordinated attack is absent from monitoring.
