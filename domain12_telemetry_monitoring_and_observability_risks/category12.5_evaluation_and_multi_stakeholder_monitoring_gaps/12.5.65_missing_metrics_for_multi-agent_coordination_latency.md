# 12.5.65 Missing Metrics for Multi-Agent Coordination Latency [NoFW]

Standard Kubernetes pod metrics capture per-pod latency but not inter-pod communication latency for multi-agent message passing and coordination; attackers can degrade inter-agent communication performance without triggering standard per-pod latency alerts, as coordination overhead is invisible to monitoring that captures only intra-pod processing time.
