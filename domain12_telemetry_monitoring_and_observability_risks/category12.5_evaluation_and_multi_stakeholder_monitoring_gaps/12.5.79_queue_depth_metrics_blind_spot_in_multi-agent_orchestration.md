# 12.5.79 Queue Depth Metrics Blind Spot in Multi-Agent Orchestration [NoFW]

Inference serving queue depth metrics reflect infrastructure queuing but not orchestration-level queuing in multi-agent workflows; when agents queue requests to other agents upstream of the inference server, orchestration-layer bottlenecks and deadlocks appear in monitoring as healthy inference queue depths that do not reflect the true multi-agent queuing state.
