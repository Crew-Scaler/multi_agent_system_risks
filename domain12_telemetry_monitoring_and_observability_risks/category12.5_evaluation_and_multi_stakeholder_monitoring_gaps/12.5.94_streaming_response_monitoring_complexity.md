# 12.5.94 Streaming Response Monitoring Complexity [NoFW]

Streaming responses in batched multi-agent contexts create boundary misalignment between batch processing units and response streaming units; per-request attribution in monitoring becomes unreliable when response boundaries do not align with batch boundaries, preventing accurate characterization of individual streaming responses' contribution to aggregate metrics.
