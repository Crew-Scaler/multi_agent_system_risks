# 12.5.59 Alert Fatigue from Pod-Level Metrics vs Service-Level Metrics [NoFW]

Kubernetes multi-agent deployments generate pod-level alerts for each agent replica; large agent fleets produce alert volumes (N agents × M replicas × P alert rules) that overwhelm operator attention, enabling adversaries to trigger numerous low-severity pod-level alerts that drown out genuine service-level anomaly signals indicating coordinated attacks.
