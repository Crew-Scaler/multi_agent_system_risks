# 12.5.112 Observability Layer Instrumentation Overhead Creating Latency Attribution Gaps [NoFW]

Distributed tracing, metrics collection, and structured logging add 50–150 ms overhead per multi-agent request chain, but monitoring systems report total latency without separating instrumentation overhead from application processing time; operators misattribute performance differences between agents to application logic differences when the actual cause is asymmetric instrumentation overhead across heterogeneously-instrumented agents.
