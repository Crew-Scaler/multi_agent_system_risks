# 12.5.96 Tool Invocation Telemetry Correlation Attacks [NoFW]

Tool telemetry systems track which agents invoke which tools, enabling pattern analysis; in multi-agent systems, attackers create specific tool invocation sequences spanning multiple agents (Agent A invokes Tool A, Agent B invokes Tool B, Agent C invokes Tool C in coordinated sequence) that encode hidden communication channels or attacker commands while appearing as legitimate per-agent tool patterns, defeating per-agent anomaly thresholds that cannot observe the cross-agent sequence as a unit.
