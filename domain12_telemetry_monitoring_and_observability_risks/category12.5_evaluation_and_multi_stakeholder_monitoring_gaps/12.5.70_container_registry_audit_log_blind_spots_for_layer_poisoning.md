# 12.5.70 Container Registry Audit Log Blind Spots for Layer Poisoning [NoFW]

Container registry logs capture image push and pull events at the image level but do not perform semantic validation of layer contents; adversaries who push container images with poisoned layers affecting agent behavior are visible in registry audit logs as normal image pushes, and in multi-agent deployments where all agents pull from the same registry, a single poisoned image affects the entire fleet without triggering content-level audit alerts.
