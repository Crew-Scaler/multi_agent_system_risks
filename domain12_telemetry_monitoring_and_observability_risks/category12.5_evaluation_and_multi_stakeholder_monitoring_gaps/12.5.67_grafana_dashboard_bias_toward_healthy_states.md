# 12.5.67 Grafana Dashboard Bias Toward Healthy States [NoFW]

Monitoring dashboards display aggregate metrics that average across agent fleet states, creating visual bias toward aggregate health; adversaries who compromise a subset of agents benefit from healthy-majority averaging that keeps aggregate dashboard indicators in normal-looking ranges while the compromised minority experiences significant degradation.
