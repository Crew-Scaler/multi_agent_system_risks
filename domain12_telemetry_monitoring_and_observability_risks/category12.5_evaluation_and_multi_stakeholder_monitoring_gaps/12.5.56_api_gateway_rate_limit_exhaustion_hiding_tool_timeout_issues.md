# 12.5.56 API Gateway Rate Limit Exhaustion Hiding Tool Timeout Issues [NoFW]

Rate limit rejections at the API gateway appear in monitoring as traffic management events, masking the underlying tool timeout failures that caused request backlog; operators investigating apparent rate limit problems are misdirected from the actual tool infrastructure failure affecting the entire multi-agent fleet that depends on the timed-out tool.
