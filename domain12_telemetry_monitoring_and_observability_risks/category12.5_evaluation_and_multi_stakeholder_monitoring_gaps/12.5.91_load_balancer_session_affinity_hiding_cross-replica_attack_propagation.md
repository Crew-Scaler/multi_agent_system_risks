# 12.5.91 Load Balancer Session Affinity Hiding Cross-Replica Attack Propagation [NoFW]

IP hash session affinity confines user sessions to specific replicas, hiding cross-replica attack propagation within load balancer routing decisions; malware propagating through routing choices is invisible in per-replica logs that show only traffic directed to each replica without surfacing how routing decisions connect cross-replica propagation chains.
