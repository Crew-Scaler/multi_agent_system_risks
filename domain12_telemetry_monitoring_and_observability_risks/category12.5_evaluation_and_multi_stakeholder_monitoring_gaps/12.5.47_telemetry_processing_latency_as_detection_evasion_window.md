# 12.5.47 Telemetry Processing Latency as Detection Evasion Window [NoFW]

Efficiency telemetry is collected and processed asynchronously with analysis occurring hours after events; adversaries executing rapid attacks finish before monitoring analysis identifies the anomaly, and in multi-agent systems where attacks can be coordinated across agents, the processing lag enables coordinated fleet-wide failures to complete and cover tracks before distributed monitoring detects anomalies across any individual agent's telemetry stream.
