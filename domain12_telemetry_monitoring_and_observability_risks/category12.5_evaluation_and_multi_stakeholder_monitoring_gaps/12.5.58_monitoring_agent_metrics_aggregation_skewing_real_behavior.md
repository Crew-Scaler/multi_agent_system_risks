# 12.5.58 Monitoring Agent Metrics Aggregation Skewing Real Behavior [NoFW]

Centralized dashboards average metrics across agent fleets; compromised agents exhibiting anomalous behavior remain undetected when their metric deviations average with normal agents' natural variance, and in large fleets where compromised agents constitute a minority, this averaging effect systematically suppresses anomaly visibility below detection thresholds.
