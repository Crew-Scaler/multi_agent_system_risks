# 12.5.88 Load Balancer Routing Metrics Obscuring Agent-Level Behavior [NoFW]

Load balancer aggregate traffic distribution metrics show that requests are spread across replicas but mask individual replica behavior; an attack affecting one replica appears as normal aggregate load distribution, and per-replica anomaly detection requires additional instrumentation beyond standard load balancer metrics.
