# 12.5.83 GPU Telemetry Insufficient for Detecting Quantization-Based Attacks [NoFW]

Standard GPU monitoring metrics (utilization percentage, memory usage, temperature, power draw) do not capture quantization-layer behavior (activation range distributions, per-layer precision loss, KV cache integrity); quantization-based instruction injection attacks that operate at the model weight precision level produce no GPU-level anomalies and remain completely invisible to DCGM-based fleet monitoring.
