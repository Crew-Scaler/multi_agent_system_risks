# 12.5.13 Evaluation Performance Monitoring Blind Spots [NoFW]

Evaluation scripts and frameworks are not subject to the same performance monitoring applied to production code; adversaries can trigger exponential resource consumption in evaluation loops (by crafting inputs that cause inefficient metric computation) to cause evaluations to time out, preventing quality gates from executing and clearing the path for unvalidated deployment.
