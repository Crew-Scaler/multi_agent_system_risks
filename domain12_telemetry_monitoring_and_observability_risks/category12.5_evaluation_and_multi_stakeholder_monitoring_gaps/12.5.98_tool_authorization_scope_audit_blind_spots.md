# 12.5.98 Tool Authorization Scope Audit Blind Spots [NoFW]

Authorization audit systems track which agents are authorized to invoke which tools, but multi-agent delegation creates accountability gaps; audit logs show "Agent A authorized to invoke Tool X" without visibility that Agent A delegated to Agent B which invoked Tool X using Agent A's authorization context, making the actual invoker appear authorized while the responsible agent was not the tool's authorized user.
