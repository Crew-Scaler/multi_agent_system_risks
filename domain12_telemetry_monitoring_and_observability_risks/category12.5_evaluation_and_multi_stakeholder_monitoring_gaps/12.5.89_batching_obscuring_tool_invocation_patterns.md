# 12.5.89 Batching Obscuring Tool Invocation Patterns [NoFW]

Tools invoked within batch processing contexts do not expose individual invocation patterns in request logs; dangerous tool call sequences that would trigger anomaly detection if visible as individual invocations are hidden within batch boundaries, requiring deep batch-internal monitoring that standard tool audit logging does not provide.
