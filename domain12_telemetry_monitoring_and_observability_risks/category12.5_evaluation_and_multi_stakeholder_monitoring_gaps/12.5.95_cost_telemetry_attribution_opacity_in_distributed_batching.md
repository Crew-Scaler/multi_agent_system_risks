# 12.5.95 Cost Telemetry Attribution Opacity in Distributed Batching [NoFW]

Distributed batch processing distributes computational cost across batch members without per-member cost attribution; agents hiding expensive operations within shared batches exploit this opacity to consume disproportionate resources without triggering per-agent cost alerts, and economic abuse patterns that would be detectable through per-request cost analysis remain invisible in batch-aggregate cost reporting.
