# 12.5.46 Historical Baseline Staleness Creating False Negatives [NoFW]

Alert thresholds based on baselines established at deployment become stale as query distributions, model behavior, and operational patterns evolve; adversaries exploit stale baselines by causing degradation that would have been anomalous at baseline establishment time but falls within the expanded variance of evolved normal operations, and in multi-agent systems with shared baselines, poisoning the shared baseline simultaneously corrupts monitoring for all agents.
