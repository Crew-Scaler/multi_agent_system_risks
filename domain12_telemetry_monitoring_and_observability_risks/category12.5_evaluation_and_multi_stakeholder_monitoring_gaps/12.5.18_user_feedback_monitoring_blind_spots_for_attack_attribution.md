# 12.5.18 User Feedback Monitoring Blind Spots for Attack Attribution [NoFW]

User feedback in multi-agent systems is attributed to the final responding agent regardless of which agent in the chain introduced the quality failure; adversaries who corrupt an intermediate agent's output can hide their impact behind last-agent attribution, preventing identification of the true source of quality degradation.
