# 12.5.12 Evaluation Metric Telemetry Spoofing [NoFW]

Evaluation metric reports transmitted over unencrypted or unauthenticated channels can be intercepted and replaced with falsified values; in multi-agent systems where all agents report to centralized telemetry, a single injection point enables an attacker to make evaluation appear to pass for the entire agent fleet simultaneously.
