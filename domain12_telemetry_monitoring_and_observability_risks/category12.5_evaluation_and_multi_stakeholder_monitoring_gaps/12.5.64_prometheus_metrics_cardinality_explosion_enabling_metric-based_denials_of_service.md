# 12.5.64 Prometheus Metrics Cardinality Explosion Enabling Metric-based Denials of Service [NoFW]

Prometheus metrics with unbounded label cardinality (labeled with user IDs, query contents, or dynamic identifiers) can be exploited by causing agents to emit high-cardinality metrics; in multi-agent fleets where all agents contribute to centralized Prometheus, coordinated high-cardinality metric emission overwhelms the monitoring system's storage capacity, causing fleet-wide monitoring blind spots.
