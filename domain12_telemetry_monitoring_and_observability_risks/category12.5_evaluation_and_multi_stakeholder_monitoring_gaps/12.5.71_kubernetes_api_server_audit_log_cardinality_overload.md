# 12.5.71 Kubernetes API Server Audit Log Cardinality Overload [NoFW]

High API call frequency in multi-agent Kubernetes deployments (frequent status updates, service lookups, ConfigMap reads across many agents) creates audit log volumes that operators disable or heavily sample to manage storage costs; adversaries who understand that detailed audit logging has been disabled or sampled route privileged operations to exploit the reduced audit coverage.
