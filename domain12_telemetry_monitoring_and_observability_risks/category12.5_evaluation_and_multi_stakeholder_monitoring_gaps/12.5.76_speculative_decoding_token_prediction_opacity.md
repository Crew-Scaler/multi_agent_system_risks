# 12.5.76 Speculative Decoding Token Prediction Opacity [NoFW]

Speculative decoding draft model predictions that are rejected by the verification step do not appear in output logs but influence final token probability distributions; downstream agents receiving output influenced by rejected draft predictions experience behavior driven by inputs that no monitoring system recorded, creating an unlogged causal factor in agent decision-making.
