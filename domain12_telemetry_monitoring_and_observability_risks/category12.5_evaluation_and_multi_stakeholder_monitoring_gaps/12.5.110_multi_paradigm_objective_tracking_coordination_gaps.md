# 12.5.110 Multi-Paradigm Objective Tracking Coordination Gaps [NoFW]

Hybrid systems track multiple paradigm-specific objectives (neural accuracy, symbolic rule coverage, learning reward signal) but lack unified objective telemetry; in multi-agent systems, no monitoring dashboard shows whether collective agent objectives remain aligned with global constraints, and attackers exploit this monitoring blindness by achieving objective misalignment where no individual agent's metric exceeds alert thresholds but collective alignment across paradigms violates specifications in ways only visible through unified cross-paradigm cross-agent objective monitoring.
