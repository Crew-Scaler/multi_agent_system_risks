# 12.5.60 Canary Metric Evaluation Blind Spots via Insufficient Sample Size [NoFW]

Canary deployments evaluate quality on sampled traffic; in multi-agent deployments where samples are distributed across many agents, per-agent sample sizes fall below statistical significance thresholds, preventing detection of agent-specific quality regressions that would be statistically significant if samples were concentrated on individual agents.
