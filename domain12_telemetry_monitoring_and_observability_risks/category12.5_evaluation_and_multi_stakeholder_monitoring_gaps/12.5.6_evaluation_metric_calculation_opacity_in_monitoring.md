# 12.5.6 Evaluation Metric Calculation Opacity in Monitoring [NoFW]

Metric values such as "accuracy: 92%" are logged without capturing the computation algorithm, ground truth definition, test case selection criteria, or which accuracy definition was applied, making it impossible to detect evaluation gaming through ambiguous metric definitions.
