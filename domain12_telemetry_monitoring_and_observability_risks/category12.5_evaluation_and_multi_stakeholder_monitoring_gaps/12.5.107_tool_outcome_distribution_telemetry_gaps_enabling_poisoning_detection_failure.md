# 12.5.107 Tool Outcome Distribution Telemetry Gaps Enabling Poisoning Detection Failure [NoFW]

Tool success rates and outcome distributions critical for expected utility calculation are not systematically monitored or logged; when attackers poison outcome distribution assumptions by modifying agent beliefs about tool success rates, monitoring has no ground truth to detect the deviation, and in multi-agent systems with shared tool registries, poisoned outcome distributions affect all querying agents simultaneously while individual agents with local tool models might detect inconsistency that shared-registry deployments cannot.
