# 12.5.30 Action Accuracy Metrics Not Captured in Production Monitoring [NoFW]

Production monitoring logs tool execution events but does not compute action accuracy metrics (parameter correctness, tool selection appropriateness) across agent chains; accuracy degradation at agent handoff boundaries goes undetected because cross-agent pipeline-level accuracy metrics are not computed, and per-agent logging shows execution without quality attribution.
