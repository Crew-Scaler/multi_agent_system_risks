# 12.5.104 Cross-Agent Streaming Correlation Failures in Distributed Tracing [NoFW]

Multi-agent systems correlating streaming traces across agents using correlation IDs face ordering failures when streaming timing variations cause events to appear out-of-order in logs; Agent A's stream events may log in different order than actual streaming due to buffering and network timing, and Agent B consuming those streams faces trace correlation confusion; attackers manipulating streaming order can fragment cross-agent traces, making causality reconstruction impossible across the N-factorial possible orderings that N concurrent streaming agents create.
