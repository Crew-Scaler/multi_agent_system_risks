# 12.5.24 Tool Invocation Observability Reduction Through Delegation [NoFW]

In hierarchical agent systems, tool invocations by worker agents do not trace directly to originating user requests; audit trails show tool invocations without the user context that authorized them, creating semantic gaps where compliance-oriented monitoring cannot verify that tool use was appropriately authorized at the request level.
