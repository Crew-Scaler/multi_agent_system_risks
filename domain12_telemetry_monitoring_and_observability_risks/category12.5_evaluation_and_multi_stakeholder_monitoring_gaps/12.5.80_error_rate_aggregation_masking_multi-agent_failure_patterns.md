# 12.5.80 Error Rate Aggregation Masking Multi-Agent Failure Patterns [NoFW]

Prometheus aggregate error rates across agent replicas and models dilute per-agent failure signals; an agent whose error rate increases from 1% to 2% in a 20-agent fleet where other agents maintain 0.1% error rates produces a fleet-wide aggregate increase from 0.1% to 0.15%, remaining below typical 5% alert thresholds while the affected agent's users experience doubled failure rates.
