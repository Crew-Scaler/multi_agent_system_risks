# 12.2.36 Confidence Score Monitoring Blind Spots From Voting Mechanism Opacity [NoFW]

Self-consistency voting distributions are collapsed to a final confidence percentage before reporting; monitoring systems cannot distinguish genuine consensus (all paths agree) from manufactured consensus (a manipulated subset of paths forces majority), as both produce the same reported confidence value.
