# 12.2.6 Tool Chain Monitoring Gaps Enabling Covert Privilege Escalation [NoFW]

Per-agent tool authorization audits verify that each agent's individual tool calls are within scope, but do not monitor cross-agent sequences; an agent with read-only database access can legitimately pass sensitive data to a second agent with email access, completing an exfiltration attack that no single-agent audit captures.
