# 12.2.28 Reasoning Mutation Defeating Anomaly Detection [NoFW]

Chain-of-thought traces can be paraphrased to preserve attack semantics while evading syntactic pattern matching; monitoring systems comparing CoT traces against known-bad signatures cannot detect semantically equivalent reformulations of malicious reasoning.
