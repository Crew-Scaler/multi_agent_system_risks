# 12.2.56 Multimodal Attribution Opacity in Tool Invocation Tracing [NoFW]

When agents invoke tools based on multimodal RAG results, monitoring cannot determine whether tools executed due to text queries, image evidence, audio context, or combinations; in multi-agent systems where Agent A's vision processing drives Agent B's tool selection, multimodal tool invocation tracing loses fidelity across modality and agent boundaries, enabling attacks that appear as legitimate multimodal reasoning in logs.
