# 12.2.53 Function Calling Anomaly Detection Evasion Through Distributed Invocation [NoFW]

Anomaly detection monitors individual agents for unusual function calling patterns such as never-before-called tools or unusual parameter values; in multi-agent systems attackers distribute anomalous invocations across multiple agents so each agent individually appears normal—Agent A occasionally invokes unusual tool X and Agent B occasionally invokes unusual tool Y—while together they execute a coordinated attack invisible to per-agent anomaly detection thresholds.
