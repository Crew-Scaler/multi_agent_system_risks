# 12.2.50 Replanning Attack Obfuscation via Normal Variation [NoFW]

MCTS replanning inherently produces variable search tree sizes, node expansion counts, and nodes-per-second rates depending on problem difficulty; adversaries exploit this by introducing subtle heuristic degradation whose monitoring signature is indistinguishable from natural performance variation, causing replanning to gradually favor attacker-preferred paths without triggering anomaly alerts.
