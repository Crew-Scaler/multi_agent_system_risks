# 12.2.54 Tool Output Validation Monitoring Gaps Across Agent Boundaries [NoFW]

Tool output validation occurs at the consuming agent level; in multi-agent systems Tool A's output validated by Agent A is not re-validated by Agent B when consumed downstream, and monitoring shows "Tool A output validated" without visibility that Agent B skipped cross-boundary re-validation, allowing corrupted tool outputs to propagate through agent chains without re-validation telemetry.
