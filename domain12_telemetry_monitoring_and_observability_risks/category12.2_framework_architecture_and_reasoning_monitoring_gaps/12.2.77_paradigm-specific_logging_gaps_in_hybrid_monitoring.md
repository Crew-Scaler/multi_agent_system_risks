# 12.2.77 Paradigm-Specific Logging Gaps in Hybrid Monitoring [NoFW]

Different hybrid paradigms generate different telemetry (neural components log embeddings and attention, symbolic components log rule firings, learning components log reward signals) and no single monitoring approach covers all paradigm-specific telemetry; attackers identify agents with incomplete paradigm monitoring coverage and target those as compromise vectors, and in hybrid multi-agent systems requiring monitoring of N agents times M paradigm types, complete coverage is operationally infeasible with current monitoring infrastructure.
