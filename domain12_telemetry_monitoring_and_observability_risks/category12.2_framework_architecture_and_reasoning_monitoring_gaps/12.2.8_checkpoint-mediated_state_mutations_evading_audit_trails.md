# 12.2.8 Checkpoint-Mediated State Mutations Evading Audit Trails [NoFW]

Agents can inject malicious state immediately before a checkpoint, causing the corrupted state to become the new baseline for subsequent restorations; audit logs record checkpoint creation and restoration events but do not capture content-level diffs or integrity hashes of checkpoint contents.
