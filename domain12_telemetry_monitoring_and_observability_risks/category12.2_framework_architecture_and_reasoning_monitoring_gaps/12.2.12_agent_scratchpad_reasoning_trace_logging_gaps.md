# 12.2.12 Agent Scratchpad Reasoning Trace Logging Gaps [NoFW]

Framework telemetry reports that an agent completed reasoning in N steps but does not log the scratchpad content of those steps, hiding injection indicators, policy drift, or manipulated intermediate conclusions that drove the final output.

## Supporting Evidence

- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
  - [→ 2602.10133.md](../../papers/2602.10133.md)
  - "Existing security methods, such as proxy-level input filtering and model glassboxing, fail to provide sufficient transparency or traceability into agent reasoning, state changes, or environmental interactions."
- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../papers/2603.11088.md)
  - "agent activity log ranges from coarse summaries of actions and tool calls to fine-grained traces that capture intermediate reasoning steps and parameters."
