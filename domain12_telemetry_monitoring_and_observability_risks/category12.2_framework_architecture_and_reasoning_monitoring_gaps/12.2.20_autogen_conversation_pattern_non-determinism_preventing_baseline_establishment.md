# 12.2.20 AutoGen Conversation Pattern Non-Determinism Preventing Baseline Establishment [NoFW]

AutoGen's non-deterministic message sequencing produces different conversation structures across executions, making static baseline pattern establishment impossible and defeating monitoring approaches that rely on comparing current behavior against historical norms.

## Supporting Evidence

- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
  - [→ 2602.10133.md](../../../papers/2602.10133.md)
  - "the inherently nondeterministic behavior of LLM agents defies static auditing approaches that have historically underpinned software assurance."
  - "Current security and auditing tools remain constrained by assumptions of determinism, procedural transparency, or bounded action space, assumptions that do not hold in LLM-based settings."
- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../../papers/2603.11088.md)

  - "Static rules miss novel attacks, whereas adaptive models incur overhead and remain susceptible to evasion."
