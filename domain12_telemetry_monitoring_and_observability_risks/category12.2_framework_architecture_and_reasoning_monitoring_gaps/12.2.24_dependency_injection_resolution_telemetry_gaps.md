# 12.2.24 Dependency Injection Resolution Telemetry Gaps [NoFW]

Kernel service resolution is not instrumented with telemetry showing which plugin resolved which service, leaving malicious dependency substitutions invisible; plugins operating with attacker-controlled service implementations execute without any logging evidence of the substitution.
