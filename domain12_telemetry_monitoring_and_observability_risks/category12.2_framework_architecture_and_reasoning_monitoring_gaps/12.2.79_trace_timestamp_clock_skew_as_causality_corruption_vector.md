# 12.2.79 Trace Timestamp Clock Skew as Causality Corruption Vector [NoFW]

Clock skew between distributed systems causes apparent impossibilities in trace ordering, with events appearing to occur before their causes. Attackers exploit this by crafting instructions that appear in traces in orders that would not be possible given true causality but seem plausible given known clock skew between agent systems. Multi-agent traces across systems with independent clocks make causality verification impossible without cryptographic proof, enabling adversarial reordering of event histories used by downstream agents for decision-making.
