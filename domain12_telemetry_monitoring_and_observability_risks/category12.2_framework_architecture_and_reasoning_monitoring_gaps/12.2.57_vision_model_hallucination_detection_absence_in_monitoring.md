# 12.2.57 Vision Model Hallucination Detection Absence in Monitoring [NoFW]

Vision models hallucinate content not present in images (NeVA hallucinating objects, DePlot hallucinating data relationships, Whisper hallucinating speech), but monitoring systems do not detect hallucinations because outputs match expected format; in multi-agent systems hallucinated content drives tool invocations and decisions without any telemetry indicating that source data was fabricated, and hallucinated outputs from one agent cascade invisibly to drive dependent agents' decisions.
