# 12.2.80 Multi-Agent Root Cause Analysis Inference Failures [NoFW]

Forensic debugging in multi-agent systems becomes unreliable when symptoms manifest in one agent but causes originate in a different agent. Attackers craft attacks with maximal separation between symptom location and cause location, exploiting the cognitive difficulty of cross-agent root cause analysis. Without distributed trace correlation linking cross-agent causality, investigators trace locally and identify the wrong causal agent, enabling attacks that create false causal trails obscuring true responsibility.
