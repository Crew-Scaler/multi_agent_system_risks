# 12.2.14 Memory Update Provenance Gaps for State Poisoning Detection [NoFW]

State mutation events record the new value but not which agent authored the change or whether the change propagated unexpectedly to other agents' shared contexts, preventing attribution of poisoned state to a specific compromised agent.

## Supporting Evidence

- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../papers/2603.11088.md)
  - "An attacker can directly manipulate the agent's memory to inject malicious instructions or false knowledge."
