# 12.2.72 Heuristic Parameter Tuning Monitoring Blindness [NoFW]

Heuristic parameters such as temperature and sampling strategies are tuned independently across agents, and monitoring cannot easily detect when parameters are tuned to enable specification gaming because changes in heuristic behavior appear as natural variation; in multi-agent systems, parameter tuning diversity creates monitoring challenges distinguishing legitimate heterogeneity from malicious parameter drift that attackers exploit knowing monitoring will miss gradual parameter changes.
