# 12.2.49 Framework-Induced Telemetry Gaps in LangGraph MCTS Integration [NoFW]

LangGraph checkpointing saves MCTS planning state (tree structure, Q values, visit counts) but monitoring systems log only checkpoint creation and restoration events without examining checkpoint content; attackers who modify saved MCTS trees between checkpoint creation and restoration corrupt future planning without generating any telemetry signal.
