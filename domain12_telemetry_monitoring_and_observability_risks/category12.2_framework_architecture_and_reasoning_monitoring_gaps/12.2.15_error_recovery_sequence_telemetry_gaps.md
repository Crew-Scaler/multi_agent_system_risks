# 12.2.15 Error Recovery Sequence Telemetry Gaps [NoFW]

LangChain error recovery logs retry count and final success status but not which code paths each retry exercised, making deliberate error injection attacks that force execution through attacker-chosen retry paths undetectable in telemetry.
