# 12.2.18 CrewAI Task Execution Trace Opacity in Hierarchical Delegation [NoFW]

CrewAI logs record which worker received a delegated task but not whether the delegation was contextually appropriate or whether the task context was manipulated between manager and worker, hiding delegation-based attack vectors.
