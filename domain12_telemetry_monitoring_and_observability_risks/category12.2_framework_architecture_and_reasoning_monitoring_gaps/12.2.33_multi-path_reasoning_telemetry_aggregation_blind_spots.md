# 12.2.33 Multi-Path Reasoning Telemetry Aggregation Blind Spots [NoFW]

Self-consistency approaches that sample multiple reasoning paths and aggregate to a majority answer report only the aggregate outcome; monitoring systems lose visibility into which individual paths contained injected instructions, as the paths collapse into a single reported answer.
