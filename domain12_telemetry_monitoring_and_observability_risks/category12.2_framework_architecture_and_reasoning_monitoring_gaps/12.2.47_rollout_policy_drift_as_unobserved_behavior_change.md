# 12.2.47 Rollout Policy Drift as Unobserved Behavior Change [NoFW]

Adaptive MCTS rollout policies that learn improved heuristics over time drift gradually without explicit monitoring; adversaries can steer rollout policy evolution toward attacker-preferred behaviors through selective input crafting while monitoring observes only executed actions with no visibility into the evolving simulation heuristics.
