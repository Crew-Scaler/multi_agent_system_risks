# 12.2.82 Real-Time Reasoning Trace Telemetry Performance Overhead Creating Sampling Bias [NoFW]

Comprehensive cross-agent reasoning trace collection creates prohibitive overhead, forcing sampling strategies; attackers exploit sampling boundaries by routing malicious operations through agents or time windows that are not sampled.
