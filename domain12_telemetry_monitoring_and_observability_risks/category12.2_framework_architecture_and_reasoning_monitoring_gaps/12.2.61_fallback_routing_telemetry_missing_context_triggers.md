# 12.2.61 Fallback Routing Telemetry Missing Context Triggers [NoFW]

Fallback routing telemetry records which fallback routes were selected but not the error conditions or context manipulation that triggered fallback; in multi-agent systems, detecting fallback attacks requires understanding trigger conditions, and telemetry lacking context trigger analysis cannot detect attacks deliberately crafting errors to force fallback to compromised alternatives.
