# 12.2.70 Rule Firing Monitoring Gaps in Distributed Systems [NoFW]

Inference traces record rule firings in rule-based systems, but in multi-agent systems with distributed rule execution across agents, monitoring rule firings across agent boundaries creates blind spots where Agent A fires rules 1–5 and Agent B fires rules 6–10 but the combined rule sequence is not visible in any single monitoring dashboard, enabling attackers to distribute malicious rule firings across agents creating patterns invisible at the individual agent level.
