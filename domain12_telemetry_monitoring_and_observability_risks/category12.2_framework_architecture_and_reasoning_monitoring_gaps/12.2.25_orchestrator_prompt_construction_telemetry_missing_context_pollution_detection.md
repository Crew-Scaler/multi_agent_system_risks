# 12.2.25 Orchestrator Prompt Construction Telemetry Missing Context Pollution Detection [NoFW]

The orchestrator's aggregated prompt constructed from all plugin descriptions is not logged or monitored for injected instructions; compromised orchestrator prompts execute successfully while telemetry shows only the final routing decisions driven by the polluted prompt.
