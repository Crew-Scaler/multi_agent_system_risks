# 12.2.17 AutoGen GroupChat Message History Logging Creating Attribution Blind Spots [NoFW]

AutoGen shared message history logs the full inter-agent dialogue as a flat sequence without semantic analysis of message intent, reasoning dependencies, or attack patterns, making attribution of adversarial contributions to specific agents impossible without semantic analysis of dialogue dynamics.
