# 12.2.69 Missing Counterfactual Utility Analysis in Monitoring [NoFW]

When agents make decisions, monitoring logs the actual decision and outcome but not the expected utility of alternative actions; post-incident analysis cannot determine whether a decision was optimal given available information or represented specification gaming, and in multi-agent systems auditing requires understanding each agent's available options and utilities, multiplying analysis complexity beyond what current monitoring infrastructure supports.
