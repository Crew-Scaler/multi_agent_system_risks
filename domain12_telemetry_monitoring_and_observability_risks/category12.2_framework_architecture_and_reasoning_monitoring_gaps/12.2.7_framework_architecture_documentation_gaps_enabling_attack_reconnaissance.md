# 12.2.7 Framework Architecture Documentation Gaps Enabling Attack Reconnaissance [NoFW]

Publicly available framework documentation describes per-framework threat models in detail, but multi-agent deployments combining multiple frameworks lack documented cross-framework integration threat models, enabling adversaries to synthesize multi-framework attack plans by composing framework-specific vulnerability knowledge.

## Supporting Evidence

- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../papers/2603.11088.md)
  - "Attackers can also target the agentic AI system itself, rather than the user or surrounding environment, by stealing system prompts, tool descriptions, and configurations that encode proprietary knowledge [137, 176], undermining agent developers and tool providers."
