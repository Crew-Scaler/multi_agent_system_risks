# 12.2.35 Sampling Parameter Monitoring Blind Spots [NoFW]

Multi-path reasoning systems use varied sampling parameters across agents and problem classes; monitoring does not track which parameter configurations were applied to each execution, creating blind spots where attacks succeed only under specific parameter regimes that are never correlated to the observed anomaly.
