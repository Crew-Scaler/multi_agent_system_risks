# 12.2.27 Framework-Specific Semantic Kernel Monitoring Blind Spots [NoFW]

Organizations standardized on LangChain or LangGraph monitoring lack instrumentation for Semantic Kernel-specific telemetry points such as plugin lifecycle events and kernel service resolution; multi-agent systems mixing Semantic Kernel with other frameworks develop permanent observability gaps at Semantic Kernel boundaries.
