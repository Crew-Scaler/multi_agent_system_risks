# 12.2.40 State Abstraction Projection Loss in Monitoring Context [NoFW]

Hierarchical monitoring agents operate at different abstraction levels; strategic agents monitor abstracted state summaries while tactical agents monitor concrete state details, and neither level has complete visibility, creating abstraction-layer gaps where problems visible only at the concrete level are invisible to agents monitoring abstract summaries.
