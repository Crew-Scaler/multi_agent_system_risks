# 12.2.30 Reasoning Latency Sidechannels in Coordination [NoFW]

Time delays between reasoning trace generation and tool invocation encode information about agent cognitive state; attackers can infer when their injection succeeded by observing characteristic latency shifts, while monitoring systems that track latency without analyzing its correlation to reasoning content miss this covert channel.

## Supporting Evidence

- Chenchen Gu, Xiang Lisa Li, Rohith Kuditipudi, Percy Liang, and Tatsunori Hashimoto. 2025. Auditing Prompt Caching in Language Model APIs. In *ICML 2025*. https://arxiv.org/abs/2502.07776
  - [→ 2502.07776.md](../../papers/2502.07776.md)
  - "These timing differences introduce the risk of side-channel timing attacks."
  - "In particular, an attacker could identify prompts that yield fast API response times; such prompts are likely cached."
- Hyejun Jeong, Mohammadreza Teymoorianfard, Abhinav Kumar, Amir Houmansadr, and Eugene Bagdasarian. 2026. Network-Level Prompt and Trait Leakage in Local Research Agents. In *USENIX Security 2026*. https://arxiv.org/abs/2508.20282
  - [→ 2508.20282.md](../../papers/2508.20282.md)
  - "Even when content is encrypted, metadata such as domain names, access order, payload size, and timing can silently reveal what a user is trying to accomplish"
