# 12.2.9 Conditional Routing Decision Opacity in Multi-Agent Monitoring [NoFW]

Conditional edge routing events record which node was selected but not the reasoning or state conditions that triggered the selection, making it impossible to determine whether routing was legitimate or the result of poisoned state manipulating control flow.
