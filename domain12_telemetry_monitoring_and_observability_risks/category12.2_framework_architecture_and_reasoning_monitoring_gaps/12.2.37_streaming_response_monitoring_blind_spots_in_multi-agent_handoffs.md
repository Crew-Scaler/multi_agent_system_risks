# 12.2.37 Streaming Response Monitoring Blind Spots in Multi-Agent Handoffs [NoFW]

When Agent A streams output directly to Agent B as input, mid-stream injection points at the agent boundary lack instrumentation; monitoring tools that capture only completed response objects miss content injected into partial streams consumed before stream completion.
