# 12.2.23 Function Description Processing Telemetry Missing Injection Detection [NoFW]

Semantic Kernel logs plugin registration events but does not analyze description content for injected instructions; poisoned descriptions that corrupt the orchestrator's prompt context leave no telemetry evidence traceable to the description manipulation.

## Supporting Evidence

- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../papers/2603.11088.md)
  - "Attackers inject malicious instructions into the names or descriptions of external tools that agents interact with."
