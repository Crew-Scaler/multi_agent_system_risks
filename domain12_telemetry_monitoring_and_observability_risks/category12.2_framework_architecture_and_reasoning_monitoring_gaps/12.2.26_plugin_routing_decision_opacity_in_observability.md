# 12.2.26 Plugin Routing Decision Opacity in Observability [NoFW]

LLM-driven plugin selection telemetry records the selected plugin but not the reasoning—which description caused selection, what alternatives were evaluated, or why a specific plugin won—preventing detection of poisoned descriptions that successfully bias routing toward attacker-preferred plugins.
