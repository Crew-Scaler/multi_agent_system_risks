# 12.2.38 Decomposition Trace Logging Gaps in Multi-Agent Hierarchies [NoFW]

HTN planning decomposition decisions—which methods were selected, what constraints were applied, and what ordering resulted—are logged within individual agents but not centrally aggregated, leaving the planning rationale invisible to system-level monitoring until execution begins.
