# 12.2.75 Gradient Flow Monitoring Limitations for Distributed Learning [NoFW]

Federated learning shares gradients between agents and monitoring gradient flows can detect anomalies; however, sophisticated attackers craft gradients appearing benign in individual inspection while encoding backdoors that activate only after aggregation, and in multi-agent systems aggregated gradients hide individual agent malice in collective statistics, defeating gradient-level monitoring that cannot separate individual agent contributions from the aggregate.
