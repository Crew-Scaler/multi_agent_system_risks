# 12.2.78 Cooperative Cycle Intermediate State Blind Spots [NoFW]

Cooperative hybrid architectures iterate through intermediate states that are not logged when monitoring focuses only on final outputs; attackers inject instructions into intermediate states causing misbehavior during processing cycles while producing acceptable final outputs, and in multi-agent cooperative monitoring spanning multiple agents, intermediate states distributed across agent boundaries lack unified visibility, creating injection opportunities at Agent A's intermediate states that affect Agent B without trace linkage between the injection and the downstream effect.
