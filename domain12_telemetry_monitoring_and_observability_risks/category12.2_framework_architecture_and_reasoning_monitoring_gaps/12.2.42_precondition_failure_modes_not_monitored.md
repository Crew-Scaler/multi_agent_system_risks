# 12.2.42 Precondition Failure Modes Not Monitored [NoFW]

HTN precondition validation performed by one agent is not independently verified by the executing agent; if the validating agent makes a precondition error, the executing agent proceeds without detecting the upstream failure, and monitoring focused on execution outcomes cannot trace resulting errors back to the precondition root cause.
