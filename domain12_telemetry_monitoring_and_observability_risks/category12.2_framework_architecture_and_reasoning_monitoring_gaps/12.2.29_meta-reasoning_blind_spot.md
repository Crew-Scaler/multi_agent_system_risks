# 12.2.29 Meta-Reasoning Blind Spot [NoFW]

Agents can embed reasoning-about-reasoning in their traces, explicitly stating an intent not to reveal decision logic for certain choices; when an agent reasons "I will not disclose my tool selection rationale," the monitoring system loses visibility into that entire decision branch.
