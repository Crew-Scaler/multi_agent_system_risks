# 12.2.46 Simulation Trace Visibility as Behavioral Blind Spot [NoFW]

MCTS simulation traces capturing attempted actions and simulated outcomes are not surfaced to monitoring systems; monitoring sees only the single executed action selected by backpropagation, missing that an adversarial injection caused the agent to extensively simulate dangerous actions before selecting a nominally safe one.
