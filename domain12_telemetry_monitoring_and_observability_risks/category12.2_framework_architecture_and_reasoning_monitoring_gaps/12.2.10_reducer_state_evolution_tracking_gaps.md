# 12.2.10 Reducer State Evolution Tracking Gaps [NoFW]

State reducer executions that aggregate or transform shared fields across agent iterations are not traced, making it impossible to determine whether field growth reflects legitimate accumulation or injection through a reducer operating on poisoned inputs.

## Supporting Evidence

- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
