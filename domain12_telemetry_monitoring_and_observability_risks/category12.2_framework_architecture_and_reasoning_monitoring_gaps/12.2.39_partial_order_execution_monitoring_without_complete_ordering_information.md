# 12.2.39 Partial Order Execution Monitoring Without Complete Ordering Information [NoFW]

When an HTN plan specifies partial ordering, executing agents log which sequence they chose without recording that alternative orderings were also permissible; monitoring systems see apparently deterministic execution while actual non-determinism provides attack surface that monitoring cannot reason about.
