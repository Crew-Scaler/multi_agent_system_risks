# 12.2.22 Plugin Execution Trace Abstraction Hiding Plugin Chain Visibility [NoFW]

Semantic Kernel orchestration summarizes multi-plugin execution as a single high-level completion event without logging the individual plugins invoked, their execution order, or parameters passed at each step, hiding malicious plugin chains from monitoring.

## Supporting Evidence

- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
  - [→ 2602.10133.md](../../papers/2602.10133.md)
  - "In tracing UIs (e.g., Jaeger), method-level operational spans appear alongside contextual spans, providing end-to-end visibility complementary to local files."
- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
