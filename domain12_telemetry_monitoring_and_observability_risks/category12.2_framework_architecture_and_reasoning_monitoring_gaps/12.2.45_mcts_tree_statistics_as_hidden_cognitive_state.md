# 12.2.45 MCTS Tree Statistics as Hidden Cognitive State [NoFW]

Monte Carlo Tree Search internal statistics (visit counts, Q values, UCT scores) represent the agent's cognitive state and planning intentions but are rarely logged; observers with access to shared logging systems, debugging APIs, or memory dumps can extract detailed planning intelligence while legitimate monitoring systems remain blind to the agent's internal deliberation.
