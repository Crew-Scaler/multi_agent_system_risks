# 12.2.11 ConversationBufferMemory Telemetry Gap for Injection Detection [NoFW]

LangChain conversation memory logs message sequences without semantic analysis of injection-indicating patterns such as suspicious tool selections, self-referential instructions, or contradictory statements, and does not capture how cross-agent message propagation carries poisoned content downstream.

## Supporting Evidence

- Chen Yueh-Han, Nitish Joshi, Yulin Chen, Maksym Andriushchenko, Rico Angell, and He He. 2026. Monitoring Decomposition Attacks in LLMs with Lightweight Sequential Monitors. In *International Conference on Learning Representations (ICLR 2026)*. https://arxiv.org/abs/2506.10949
  - [→ 2506.10949.md](../../../papers/2506.10949.md)
  - "Modern safety-tuned LLMs fail under decomposition attacks due to their shallow alignment: they detect harm and refuse only when it is explicit in the immediate prompt, and the model does not have to reason about long-range harmful intent."
- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
  - [→ 2602.10133.md](../../../papers/2602.10133.md)
  - "These approaches offer limited insight into agent intent, decision provenance, or operational context, especially in the presence of tool-use chaining, memory operations, or multi-agent collaboration."
- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../../papers/2603.11088.md)
  - "An attacker can directly manipulate the agent's memory to inject malicious instructions or false knowledge."
