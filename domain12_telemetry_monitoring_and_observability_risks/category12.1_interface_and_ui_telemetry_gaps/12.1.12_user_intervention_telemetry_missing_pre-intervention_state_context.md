# 12.1.12 User Intervention Telemetry Missing Pre-Intervention State Context [NoFW]

Human override events record that an intervention occurred but not the distributed agent state at the moment of intervention, preventing learning from intervention patterns or determining whether an intervention blocked an attack or cancelled legitimate work.
