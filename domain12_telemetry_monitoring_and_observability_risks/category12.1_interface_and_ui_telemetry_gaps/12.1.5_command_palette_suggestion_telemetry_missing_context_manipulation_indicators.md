# 12.1.5 Command Palette Suggestion Telemetry Missing Context Manipulation Indicators [NoFW]

AI-driven command palette suggestions are logged by outcome (which command was executed) but not by context provenance (which signals triggered the suggestion), making it impossible to detect when poisoned cross-agent context causes malicious commands to surface as legitimate suggestions.

## Supporting Evidence

- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../papers/2603.11088.md)
  - "For monitoring, web agents emit browsing telemetry for post-hoc review, yet none of the surveyed systems pair these logs with automated detection."
