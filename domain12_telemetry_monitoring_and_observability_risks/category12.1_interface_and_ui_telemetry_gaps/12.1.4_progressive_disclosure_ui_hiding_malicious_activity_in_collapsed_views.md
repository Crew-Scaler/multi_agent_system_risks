# 12.1.4 Progressive Disclosure UI Hiding Malicious Activity in Collapsed Views [NoFW]

Progressive disclosure UI patterns hide technical details and reasoning traces in collapsed views; monitoring focuses on user-visible essential layers while malicious agent operations execute in collapsed technical views that are rarely inspected, creating per-agent observability blind spots that compound in multi-agent systems.
