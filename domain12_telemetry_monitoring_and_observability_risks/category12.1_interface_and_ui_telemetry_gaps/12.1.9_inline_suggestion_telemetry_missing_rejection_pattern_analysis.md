# 12.1.9 Inline Suggestion Telemetry Missing Rejection Pattern Analysis [NoFW]

Telemetry captures suggestion acceptance rates but not rejection events with their context or reason, preventing detection of compromised agents that systematically suggest dangerous patterns which users consistently reject.

## Supporting Evidence

- Adam AlSayyad, Kelvin Yuxiang Huang, and Richik Pal. 2026. AgentTrace: A Structured Logging Framework for Agent System Observability. In *AAAI 2026 Workshop on Language Model Agent Systems (LaMAS)*. https://arxiv.org/abs/2602.10133
