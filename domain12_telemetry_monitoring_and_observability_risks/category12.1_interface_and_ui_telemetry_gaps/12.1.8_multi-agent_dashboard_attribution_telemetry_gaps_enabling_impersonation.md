# 12.1.8 Multi-Agent Dashboard Attribution Telemetry Gaps Enabling Impersonation [NoFW]

Dashboard telemetry records which agent label generated content but does not capture whether that agent's identity was verified at each handoff point, making impersonation attacks and identity spoofing invisible to audit.

## Supporting Evidence

- Juhee Kim et al.. 2026. The Attack and Defense Landscape of Agentic AI: A Comprehensive Survey. In *USENIX Security 2026*. https://arxiv.org/abs/2603.11088
  - [→ 2603.11088.md](../../papers/2603.11088.md)
  - "Foundational questions persist about whether agent actions should be attributed to the human operator, the agent instance, or transient task identities, and the answer often varies across domains."
