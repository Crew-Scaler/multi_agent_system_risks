# 12.1.6 Error Recovery Telemetry Treating Retry Attempts as Atomic Events [NoFW]

Automatic retry mechanisms log each attempt as an independent event rather than as a sequence, preventing detection of attacks that deliberately trigger errors to force specific retry code paths or that exploit different security contexts across retry attempts.
